## 3. 아키텍처 (요약)
### 3.1 데이터
- Hot Tier: PostgreSQL 16 (RW)
- Warm Tier: PostgreSQL Replica + Partition
- Cold Tier: S3 + Glacier (WORM, 5년)
- 이벤트 버스: Kafka 3.5

### 3.2 통신
- 사용자↔Gateway: HTTPS REST (JSON)
- 내부 마이크로서비스: gRPC + Kafka
- ERP 외부: REST (실시간) + AS2 (배치)

### 3.3 보안
- 인증: OIDC (Okta), MFA 강제
- 인가: RBAC + 라인 단위 scope
- 암호화: AES-256-GCM (저장), TLS 1.3 (전송), HSM (키)
- 감사: 모든 PII 접근 로그, 5년 보존
