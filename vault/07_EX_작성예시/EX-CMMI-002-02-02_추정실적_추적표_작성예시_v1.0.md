# 추정치 대비 실적 추적표 작성예시 (EX-CMMI-002-02-02)

> 가상 "디지털콜 v3" 기준.

## 1. 헤더
| 프로젝트 | 디지털콜 v3 |
|---|---|
| 보고 주기 | 월간 (2026-06) |

## 2. 추정 vs 실적
| 항목 | 단위 | 추정 | 실적 | 편차 | 비고 |
|---|---|---|---|---|---|
| 일정(누적) | 영업일 | 88 | 92 | +4.5% | API 합의 지연 |
| 작업량 | MM | 36 | 38 | +5.6% | 외주 추가 |
| 비용 | M원 | 380 | 405 | +6.6% | 외주 비용 |
| 규모 | 기능점수 | 240 | 252 | +5% | 추가 요구 5건 |

## 3. 편차 원인 분석
- 주요 원인: 외부 결제사 API 합의 지연으로 외주 1MM 추가 투입
- 시정조치: ISS-2026-022 → 6/9 합의 완료, 후속 지연 방지

## 작성 요령
- 편차 (%) = (실적-추정)/추정 ×100
- 단위 일관 유지.

## 잘못된 사례
- 단위 불일치 (시간/일/주 혼용).
- 편차 0% 보고 — 실측 의심.
