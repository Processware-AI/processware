# 측정 및 분석방법 정의서 작성예시 (EX-CMMI-02-04-03-01)

## 1. 측정 정의
| ID | 명칭 | 정의식 | 단위 | 주기 | 출처 |
|---|---|---|---|---|---|
| M-001 | 결함누수율 | UAT후결함 / 총결함 ×100 | % | 월 | JIRA |
| M-002 | 일정편차 | (실적-계획)/계획 ×100 | % | 주 | MS Project |
| M-003 | 요구변경률 | 변경요구수 / 총요구수 ×100 | % | 월 | JIRA |
| M-004 | 결함밀도 | 검토결함수 / KLOC | 건/KLOC | 스프린트 | SonarQube + JIRA |

## 2. 분석 방법
| ID | 기법 | 시각화 | 보고 |

| M-003 | 일별 누적 | Bar | 월 |
| M-004 | 검토 횟수당 | Box plot | 스프린트 |
