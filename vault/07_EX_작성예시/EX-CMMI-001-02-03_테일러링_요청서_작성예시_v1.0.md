# 테일러링 지침서 작성예시 (EX-CMMI-001-02-03)

> 원본 양식: [[TMP-CMMI-001-02-03_테일러링_요청서_v1.0]]

> 이 파일은 **교육·가이드용 샘플** 입니다. 가상 프로젝트 "디지털콜 v3" 기준.

## 1. 헤더
| 항목 | 내용 |
|---|---|
| 적용 범위 | 전사 SW 개발 프로젝트 |
| 발효일 | 2026-05-01 |
| 차수 | v1.0 |

## 2. 테일러링 가능 항목
| 영역 | 테일러링 옵션 | 제약 조건 |
|---|---|---|
| 동료검토 회수 | 1~3회/스프린트 | 최소 1회 / 산출물 |
| 측정 주기 | 주간 / 격주 | 분기 보고 시 합산 가능해야 함 |
| 형상감사 주기 | 분기 / 반기 | 주요 릴리스 직전 1회 필수 |

## 3. 테일러링 금지 항목
| 항목 | 사유 |
|---|---|
| 변경관리 CCB 승인 | 변경통제 핵심 — ML3 평가 직결 |
| 부적합 식별·기록 | PQA 의무 Practice |
| 양방향 추적성 작성 | RDM 2.4 필수 |

## 4. 테일러링 신청·승인 절차
| 단계 | 책임자 | 입력 | 출력 |
|---|---|---|---|
| 1. 신청 | PM | 테일러링 신청서 | OSSP-TR-{YYYY}-{###} 등록 |
| 2. 검토 | SEPG | 사유·영향분석 | 검토 의견서 |
| 3. 승인 | PCB | 검토 의견 | 결재 + PAL 등재 |

## 5. 테일러링 기록 보존
| 항목 | 위치 | 보존 기간 |
|---|---|---|
| 신청서 | PAL §테일러링 | 5년 |
| 승인 결과 | PAL §테일러링 + 프로젝트 폴더 | 5년 |

## 6. 결재
| 작성 | 검토(PCB) | 승인(CEO) |
|---|---|---|
| 김SEPG (2026-04-20) | PCB 의장 (2026-04-25) | 이CEO (2026-04-29) |

## 작성 요령
- 옵션은 정량 표현(횟수·기간·조건) 사용.
- 금지 항목은 ML3 평가 핵심 Practice 와 동기.
- "검토자 판단" 같은 모호 표현 금지.

## 잘못된 작성 사례
- "프로젝트 상황에 따라 자유롭게 조정" — 통제 불가.
- 금지 항목 칸을 비워둠 — 표준 일탈 위험.
