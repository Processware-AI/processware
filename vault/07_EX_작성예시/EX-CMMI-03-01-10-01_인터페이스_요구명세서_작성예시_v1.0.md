# 인터페이스 요구사항표 작성예시 (EX-CMMI-03-01-10-01)

> 원본: [[TMP-CMMI-03-01-10-01_인터페이스_요구명세서_v1.0]]

## 1. IF 정의
| IF | 송신 | 수신 | 프로토콜 | 데이터 | 빈도 |
|---|---|---|---|---|---|
| IF-01 | router-svc | notify-svc | gRPC | RouterEvent | 실시간 |
| IF-02 | core | 외부 PG-X | REST/JSON | PaymentReq | 트랜잭션 단위 |
| IF-03 | DB | DW | 배치 SQL | 통화 로그 | 일 1회 |

## 2. SLA
| IF | SLA | 오류 |
|---|---|---|
| IF-01 | < 100ms P95 | retry 3회 → DLQ |
| IF-02 | < 2s | exponential backoff |
| IF-03 | 02:00~03:00 | 다음날 보정 |
