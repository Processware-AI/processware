## 2. 평가 방법 후보·선정 (샘플)
| 방법 후보 | 적합도 평가 | 채택 여부 |
|---|---|---|
| 가중합산 | 객관 점수 산출 + 가중치 모델 | ✅ 주 방법 |
| AHP | 비교 행렬 복잡, 6 기준 × 4 대안 → 가능하나 일관성 비율 관리 필요 | △ 보조 (1순위 검증용) |
| PoC/실험 | 처리량·지연 측정에 필수 | ✅ 정량 기준 입력 |
| 시뮬레이션 | 부하 시뮬레이터 사용 | ✅ PoC 부족분 보완 |
| 시나리오 분석 | 운영 시 장애 시나리오 분석 | △ 운영 안정성 기준 보조 |

| 항목 | 내용 |
|---|---|
| 선정 방법(또는 조합) | **가중합산(주) + PoC/실험(정량) + 시나리오 분석(보조)** |
| 선정 근거 | 객관성: 가중합산이 점수 산출 명확. 재현성: PoC 결과를 평가 매트릭스 입력으로 사용. 비용: AHP 단독은 일관성 행렬 작성 부담 — 가중합산이 효율적 |

## 작성 시 유의사항
- 단일 방법보다 조합이 객관성·재현성에 우수한 경우가 많음.
- 정량 기준은 PoC/실험으로 측정, 정성 기준은 패널 평가 + 시나리오 분석.
- 비용·일정 제약 명시.
