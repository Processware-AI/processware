## 샘플 컨텍스트

| 버전 | 1.0 | 신규 1.0 |
| 프로젝트/조직 | NEXT 플랫폼 v3.0 / 알파소프트 | 정식 명칭 |
| 작성자 | 박OO (CM Manager) | 이름+직책 |
| 작성일 | 2026-04-20 |  |
| 검토자 | 이OO (PM), 정OO (Lead Architect) |  |
| 승인자 | 최OO (CCB Chair) | CCB 위원장 |

## 작성 시 유의사항
- 모든 CI 는 반드시 단일 Owner 를 가짐 (책임 분산 금지).
- CI ID 명명규칙은 분류·연속번호 조합으로 일관성 유지.
- 비CI 도 사유와 함께 기록하여 누락이 아니라 의도된 제외임을 증빙.
