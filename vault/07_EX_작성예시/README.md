# 07_EX_작성예시 — 작성예시(EX)

## 역할
템플릿을 어떻게 작성해야 하는지 보여주는 **교육용 샘플** 문서.

## 특징
- 샘플 데이터 포함
- 교육용/가이드용
- **실제 기록으로 사용 금지** → 실제 기록은 `08_REC_기록/`
- TMP 와 **반드시 분리 보관**

## 파일명 규칙
```
EX-{기능}-{###}_{이름}_작성예시_v{버전}.md
```
예: `EX-IR-001_침해사고_내부신고서_작성예시_v1.0.md`

## Frontmatter 필수
`type`, `doc_id`, `title`, `parent_tmp`

## 포함 항목
- 샘플 입력값 (현실적인 예시)
- 작성 요령
- 잘못된 작성 사례 (주석·비교)
- 작성 시 유의사항

## 판단 기준
> "문서 안에 **예시 값이 채워져 있으면** EX"

## 생성 담당
Agent `wi-tmp-writer` (TMP 생성 시 각 TMP 당 1건 동시 생성)

## 금지사항
- ❌ 실제 기록 데이터 포함 (개인정보·고객정보)
- ❌ TMP 에 직접 기입 (반드시 별도 파일)

## 관련 폴더
- 원본 양식: `06_TMP_템플릿/`
- 실제 기록: `08_REC_기록/` (운영 단계)
