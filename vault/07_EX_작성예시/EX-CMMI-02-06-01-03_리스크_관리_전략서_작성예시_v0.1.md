# 리스크 관리 전략서 작성예시

> 원본: [[TMP-CMMI-02-06-01-03_리스크_관리_전략서]]

## 1. 문서 정보 (샘플)
| 항목 | 내용 |
|---|---|
| 프로젝트명 | 모바일 헬스케어 앱 v2.0 |
| 버전 | 1.0 |
| 작성자 | 박OO (Risk Manager) |
| 작성일 | 2026-05-15 |

## 2. 식별·분석·완화 절차 (샘플)
| 단계 | 활동 | 빈도 |
|---|---|---|
| 식별 | brainstorming + PP/PMC/IPM 회의록 검토 + ad-hoc | 주 1회 정기 + 상시 |
| 평가 | 확률·영향·우선순위 부여 | 식별 후 24h 내 |
| 완화 | High/Critical 완화 계획 수립·실행 | 평가 후 1주 |
| 모니터링 | status 갱신 | 주 1회 |

## 3. RACI 매트릭스 (샘플)
| 활동 | Risk Mgr | PM | Engineer | Senior Mgmt |
|---|---|---|---|---|
| 식별 | R | C | C | - |
| 평가·분류 | R | C | C | - |
| 완화 계획 | R | C | C | A (Critical) |
| 완화 실행 | C | R | C | A |
| 모니터링 | R | C | - | I (월간) |

## 4. 도구·저장소 (샘플)
| 항목 | 내용 |
|---|---|
| 리스크 도구 | Jira (custom issue type: Risk) + 주간 Slack 알림 |
| 등록부 저장 위치 | Confluence space (HCA-Risk) |
| 보고 채널 | 주간 PMO 보고, 마일스톤 Senior Mgmt 보고 |

## 5. Senior Mgmt 결재 (샘플)
| 검토 | 승인 | 일자 |
|---|---|---|
| 김OO (PM), 이OO (PMO) | 최OO (CTO) | 2026-05-16 |

## 작성 시 유의사항
- 절차·빈도·도구 모두 명시 — 일회성 문서가 아닌 운영 가이드 역할.
