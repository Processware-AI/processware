# 공급자 합의서 작성예시 (EX-CMMI-005-02-02)

> 원본 양식: [[TMP-CMMI-005-02-02_공급자_합의서_v1.0]]
> "디지털콜 v3" 외부 메시지 큐 도입 합의 샘플(가공).

## 1. 헤더
| 항목 | 내용 |
|---|---|
| 합의 ID | SA-DC-2026-005 |
| 발주자 | (당사) DC Corp. |
| 공급자 | XCloud Korea Ltd. (가상) |
| 체결일 | 2026-06-30 |
| 유효 기간 | 2026-07-01 ~ 2027-06-30 |

## 2. 인수 범위·인도물
- 메시지 큐 클라우드 서비스 (Managed SQS-호환) 12개월 운영.
- 인도물: SLA 99.95% 가동, 이중화 운영, 모니터링 대시보드.

## 3. 마일스톤·결제 조건
| 마일스톤 | 인도물 | 일정 | 결제(%) |
|---|---|---|---|
| 셋업 완료 | 환경 + IAM | 2026-07-15 | 20 |
| 통합 인수 검증 | API 호환성·성능 | 2026-08-15 | 30 |
| 운영 시작 | 운영 가동 | 2026-09-01 | 30 |
| 종료 인계 | 12개월 종료, 데이터 인계 | 2027-06-30 | 20 |

## 4. SLA·페널티
| 지표 | 목표 | 위반 시 페널티 |
|---|---|---|
| 가동률 | ≥ 99.95% | 0.05%당 월 결제액 1% 차감 |
| 평균 응답 | ≤ 50ms | 100ms 초과 시 1% 차감 |
| 보안 사고 | 0건 | 사고 1건당 50,000천원 |

## 5. 보안·NDA
- ISO/IEC 27001 인증 보유 필수.
- 사고 발생 시 1시간 내 통보, 24시간 내 격리.

## 6. 분쟁 해결
- 1단계: 양사 분쟁조정위원회 30일 협의.
- 2단계: 대한상사중재원 중재.

## 7. 종료·인수 책임 이전
- 종료 30일 전 데이터 인계 시작, 종료일까지 운영 책임 공급자.
- 종료 후 데이터 안전 삭제 인증서 제출 의무.

## 8. 결재
| 발주자 | 공급자 | 법무 |
|---|---|---|
| 정구매 (2026-06-30) | 김XCloud (2026-06-30) | 한법무 (2026-06-28) |

## 작성 요령
- SLA 는 측정 정의 일치 필수(가동률 정의 등 부속서).
- 페널티는 "비율 + 절대값" 양쪽 보장.

## 잘못된 작성 사례
- 정의 모호한 "최선" SLA — 분쟁 시 무력
- 페널티 한도 무한정 — 공급자 거절
- 종료 시 데이터 인계 절차 없음 — 락인 위험
