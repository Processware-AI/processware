## 입력 통계
- **총 PDF 수**: 40 (영문 20 + 한글 20)
- **PA 수**: 20 (Core 17 + Development 2 + Supplier 1)
- **라이선스**: 전부 ISACA/CMMI Institute 자료 (paraphrase 강제)
- **MD 변환본**: 없음 (PDF only) — 페이지 지정 Read 필수

## CMMI-DEV-ML3 범위
- **Maturity Level**: 3 (= ML2 PA 전부 + ML3 PA 추가 평가)
- **ML2 PA**: CM, MC, MPM, PAD, PLAN, PQA, RDM, RSK, SAM (Practice Group 1~2)
- **ML3 PA**: 위 + CAR, DAR, EST, GOV, II, OT, PCM, PR, VV, PI, TS (Practice Group 3 까지 평가)
- 본 편입은 **ML3 평가 통과 가능 수준** 의 POL/PRO/WI 체계를 목표로 함.

## 저작권 가드레일
- 모든 PA PDF는 ISACA/CMMI Institute 저작권 보호 자료로 간주
- **20단어 이상 연속 일치 금지** ([[05_입력자료_규칙]] §6)
- 본문 인용은 `paraphrase only`. 조항·Practice Number 인용은 허용
- qa-reviewer 가 자동 검사
