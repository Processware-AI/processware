# 03_해설서

## 들어가는 것
- 인증기관(KFQ·BSI·DNV·SGS 등) 발간 **실무 가이드**
- 심사원 교재·ISO 해석 매뉴얼
- 한국표준협회 해설 자료
- ISO/TS 류의 **공식 해설 문서** (예: ISO/TS 9002 — ISO 9001 해설)

## 예시
```
03_해설서/
├── KFQ_ISO9001_실무가이드_2023.pdf
├── BSI_27001_심사_체크리스트.pdf
├── ISO_TS_9002_해설.pdf
└── 한국표준협회_품질경영시스템_해설.pdf
```

## 들어가지 않는 것
- 표준 원본 → `01_표준원문/`
- 법규·공공 고시 → `02_법규/`
- 고객사 내부 매뉴얼 → `04_AsIs/`

## 저작권 주의
- **발행처마다 라이선스 상이** — 반드시 확인
- 인증기관 무료 공개 자료: paraphrase 인용 가능
- 상용 교재: 내부 참조만, 직접 인용 금지
- ISO/TS: ISO 저작권 — paraphrase only

## 활용 가치
- 표준 원문만으로는 해석이 모호한 조항의 **실무 적용 사례**
- 심사원이 실제로 보는 **체크리스트·지적 사례**
- 업종별 **해석 관행**

## 변환 권장
- **paraphrase MD** 가 이상적 (저작권+효율)
- 핵심 해석·체크리스트만 발췌

## 에이전트 참조 우선순위
`01_표준원문` 다음. **법규·표준에 규정이 없는 세부 구현**에 대한 힌트로 사용.
