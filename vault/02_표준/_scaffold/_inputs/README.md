### 기입 가이드
- **우선순위**: ⭐⭐⭐ 핵심 / ⭐⭐ 중요 / ⭐ 참고
- **content_mode**: `verbatim` | `paraphrase` | `summary` (MD 변환본만)
- **라이선스**: `공공저작물` | `ISO copyright — paraphrase only` | `내부 자산` | `[확인 필요]`

## 편입 전 체크리스트
- [ ] 최소 1개 카테고리에 파일 배치 (없으면 LLM 추정 모드)
- [ ] 라이선스 상태 명시 (`[확인 필요]` 포함)
- [ ] 큰 PDF(>50페이지)는 MD 변환본 병기 권장
- [ ] 저작권 보호 자료(ISO 원문)는 `content_mode: verbatim` 으로 표기
