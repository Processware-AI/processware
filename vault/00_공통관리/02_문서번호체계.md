# 문서번호 체계

## 기본 파일명 형식
```
[유형코드]-[식별번호]_[문서명]_v[버전].md
```

## 유형별 식별번호 규칙

| 유형 | 식별번호 규칙 | 예시 |
|---|---|---|
| POL | `POL-{영역}-{###}` | `POL-QMS-001_품질방침_v1.0.md` |
| PRO | `PRO-{영역}-{###}` | `PRO-ISMS-101_정보보안_총괄_절차_v1.0.md` |
| WI  | `WI-{상위PRO번호}-{##}` | `WI-101-02_문서_검토_및_승인_v1.0.md` |
| TMP | `TMP-{기능}-{###}` | `TMP-IR-001_침해사고_내부신고서_v1.0.md` |
| EX  | `EX-{TMP번호}` | `EX-IR-001_침해사고_내부신고서_작성예시_v1.0.md` |
| REC | `REC-{기능}-{YYYY}-{###}` | `REC-IR-2026-001_침해사고_신고서.md` |
| MAT | `MAT-{###}` (3자리 고정) | `MAT-002_규제요구사항_대조표_v1.0.md`, `MAT-006_{표준코드}_추적성_v1.0.md` |
| REF | `REF-{###}` | `REF-001_개인정보보호법_요약_v1.0.md` |

## MAT 번호 할당 원칙

MAT 는 3자리 순차 번호로, 역할에 따라 구역을 나누어 관리한다.

| 구역 | 용도 | 할당 규칙 |
|---|---|---|
| **MAT-001 ~ MAT-005** | 전사 공통 관리대장 (고정 5종) | 절대 재배정 금지 |
| **MAT-006 ~ MAT-099** | 표준별 추적성 매트릭스 | 표준 편입 순서대로 순차 부여 |
| **MAT-100 이후** | 예약 (통합 뷰·교차 매트릭스 등) | 필요 시 규칙 합의 후 사용 |

### 전사 공통 5종 (고정)
| 번호 | 문서 |
|---|---|
| MAT-001 | 문서관리대장 |
| MAT-002 | 규제요구사항 대조표 |
| MAT-003 | 산출물 목록표 |
| MAT-004 | RACI 통합표 |
| MAT-005 | 심사증적 인덱스 |

### 표준별 추적성 (순차)
파일명: `MAT-{###}_{표준코드}_추적성_v{버전}.md`

| 번호 | 표준 | 편입일 |
|---|---|---|
| MAT-006 | (첫 편입 표준) | — |
| ... | | |

`traceability-mapper` 는 표준 편입 시 **MAT-006 부터 순차적으로 다음 미사용 번호**를 부여한다. 번호 재사용 금지 — 표준이 삭제되어도 그 번호는 결번 처리.

## 버전 규칙
- **v0.x** : 초안(draft) — 미승인
- **v1.0** : 최초 승인판
- **v1.n** : 경미 개정 (오탈자·링크·소절 수정)
- **v{n+1}.0** : 본문 구조/책임/범위 변경
- **REC 는 버전 불필요** (기록은 수정 통제 대상, 재발행 시 신규 REC로)

## 개정 절차 연계
모든 개정은 [[MAT-001_문서관리대장]] 에 기록. 상세 절차는 [[WI-102-04_개정_및_버전관리]].
