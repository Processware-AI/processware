# 측정 활동 평가 및 개선 (WI-CMMI-02-04-08)

> 상위 절차: [[PRO-CMMI-02-04_성과_및_측정_관리_절차_v1.0]]

## 1. 업무 목적
측정 활동 자체가 의사결정에 도움이 되는지 평가하고 개선한다.

## 2. 수행 주체
- 주 수행자: MPM Lead
- 검토자: PCB
- 승인자: CEO

## 3. 범위
- 반기 1회

## 4. 입력 / 산출물
- Input: 측정 활동 비용·효과 데이터
- Output: [[TMP-CMMI-02-04-08-01_측정_활동_평가서_v1.0]]

## 5. 수행 절차

### 5.1 사전 준비
1. 평가 항목 정의.
2. 사용자 의견 수집.

### 5.2 수행 단계
1. **활용도 분석**.
2. **비용·효과 분석**.
3. **만족도 조사**.
4. **격차 식별**.
5. **개선 권고**.
6. **이행 추적**.

### 5.3 완료 조건
- [ ] 평가서 승인본
- [ ] 권고사항 등록
- [ ] 이행 추적 시작

## 6. 인터페이스 부서
- PCB, 모든 사용자

## 7. 주의사항 / 예외 처리

### 7.1 사용 저조 KPI
- 폐기 또는 재정의.

### 7.2 비용 과다
- 자동화 검토.

### 7.3 만족도 저조
- UI/UX 개선.
