# 부적합 의사소통·해결·기록 (WI-CMMI-04-03-02)

> 상위 절차: [[PRO-CMMI-04-03_프로세스_제품_품질보증_절차]] · 표준: CMMI-DEV V1.3 PPQA SG2

## 1. 업무 목적
객관적 평가에서 발견된 부적합·관찰사항을 Process Owner·이해관계자에게 의사소통하고, 시정조치를 추적·종결하며, 미해결 시 경영진에 에스컬레이션하여 객관적 통찰을 제공한다.

## 2. 수행 주체
- **주 수행자**: QA Engineer (PPQA) — 핸드오프·추적
- **공동 책임자**: Process Owner — 시정조치 실행
- **승인자**: Senior Mgmt (에스컬레이션 시)

## 3. 범위
PRO-CMMI-04-03 §5 의 **SP2.1(부적합 의사소통·해결) ~ SP2.2(기록)** 적용. [[WI-CMMI-04-03-01]] 핸드오프 입력 필수.

## 4. 입력 자료 / 산출물
- **Input**
  - 부적합 목록 ([[WI-CMMI-04-03-01]] 산출)
  - 형상감사 부적합 ([[WI-CMMI-04-01-03]] 산출, 해당 시)
- **Output**
  - [[TMP-CMMI-04-03-02-01_QA_시정조치_보고서]]
  - [[TMP-CMMI-04-03-02-02_품질_트렌드_및_평가로그]]

## 5. 수행 절차

### 5.1 사전 준비
1. 평가 보고서·부적합 목록 확보.
2. Process Owner·이해관계자 연락 채널 확인.
3. 시정조치 추적 대장 최신화.

### 5.2 수행 단계
1. **부적합 의사소통 (SP2.1 전반)**
   - 부적합을 Process Owner 와 Project Manager 에 통보.
   - 부적합 등급(Critical/Major/Minor) 부여.
   - 통보 일시·방법·수신자 기록.
2. **시정조치 계획 수립**
   - Process Owner 가 근본원인 분석(5-Why·Fishbone 등) 수행.
   - 시정조치 계획(액션·책임자·기한) 작성.
   - QA Engineer 가 적정성 검토.
3. **시정조치 실행·검증**
   - Process Owner 가 액션 실행.
   - QA Engineer 가 종결 증적 확인 (재현 테스트·문서 갱신 확인).
   - 종결 또는 재발행 결정.
4. **에스컬레이션 (미종결 시)**
   - 기한 초과 또는 합의 불가 시 Senior Mgmt 에 에스컬레이션.
   - 에스컬레이션 메모 작성·결재.
5. **기록 작성 (SP2.2)**
   - 평가 로그·시정조치 결과를 누적 기록.
   - 분기별 품질 트렌드 분석 (부적합률·평균 종결 일수·에스컬레이션 비율).
6. **이해관계자 보고**
   - 분기 보고서를 경영진·PM·QA Director 에 배포.
   - OPF([[PRO-CMMI-01-02]]) 개선 안건 후보로 등록.

### 5.3 완료 조건
- [ ] 모든 부적합이 Process Owner·PM 에 통보·기록
- [ ] 시정조치 계획서 작성·검토 완료 (액션·책임자·기한)
- [ ] 종결 증적 확인 후 종결 처리
- [ ] 기한 초과 부적합 = 0 또는 에스컬레이션 처리
- [ ] 평가 로그 갱신·분기 트렌드 분석 완료
- [ ] 경영진·PM 분기 보고 완료

## 6. 인터페이스 부서
- **PPQA**: 본 지침 주 수행
- **Process Owner**: 시정조치 실행 (R)
- **Project Manager**: 부적합 인지·일정 영향 관리
- **Senior Mgmt**: 에스컬레이션 결재
- **OPF**: 개선 안건 환류 ([[PRO-CMMI-01-02]])
- **CM**: 무단 변경 발견 부적합 입력 ([[WI-CMMI-04-01-03]])

## 7. 주의사항 / 예외 처리

### 7.1 부적합 이의 제기 (Process Owner)
- Process Owner 가 부적합 판정에 이의 시:
  - 평가 증적 공동 검토 → 합의 시도.
  - 미합의 시 QA Director 가 중재 → 최종 판정.

### 7.2 시정조치 기한 초과 (30일)
- 30일 미종결 시 자동 에스컬레이션.
- 90일 미종결 시 경영진 KPI 영향 (POL §통제점).

### 7.3 동일 부적합 반복 발생
- 동일 부적합이 2회 이상 반복 시 근본원인 미해결로 간주:
  - OPF 개선 안건으로 정식 등록.
  - 차상위 시정조치(시스템 개선) 권고.
