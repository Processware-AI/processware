# 교육 효과성 평가 (WI-CMMI-01-03-03)

> 상위 절차: [[PRO-CMMI-01-03_조직_훈련_절차]]

## 1. 업무 목적
완료된 교육이 조직·프로젝트·개인의 학습·수행 목표를 달성했는지 사전·사후 평가·현장 관찰·만족도·강사 평가 등 다각도로 측정하여, 결과를 차기 전술계획·커리큘럼 갱신·전략 요구 식별에 환류한다.

## 2. 수행 주체
- **주 수행자**: Training Manager
- **검토자**: EPG Lead, Process Owner
- **승인자**: CEO/CTO (반기 효과성 보고 승인)

## 3. 범위
PRO-CMMI-01-03 §5 의 **SP2.3** 단계. 결과는 차기 사이클 [[WI-CMMI-01-03-01]] SP1.1 입력으로 환류.

## 4. 입력 자료 / 산출물
- **Input**
  - 이수 기록부·만족도 데이터 ([[WI-CMMI-01-03-02]] 산출)
  - 사전·사후 평가 결과
  - 강사 평가서·교육 시험 결과
  - 현장 관찰·인터뷰 (선택)
  - 프로젝트 성과 데이터 (MA 측정저장소 연동)
- **Output**
  - [[TMP-CMMI-01-03-03-01_교육_효과성_평가_보고서]] 작성본 (반기)
  - 차기 사이클 입력 메모

## 5. 수행 절차

### 5.1 사전 준비
1. 평가 대상 과정 선정 (반기 단위, 우선순위 과정 위주).
2. 평가 차원 정의 (Kirkpatrick 1~4 단계 추천: 반응·학습·행동·결과).
3. 사전 평가 결과 확보 (교육 시작 전 측정 데이터).

### 5.2 수행 단계
1. **반응 평가** (Kirkpatrick L1)
   - 만족도 설문 집계 ([[WI-CMMI-01-03-02]] 산출).
   - 강사·교재·일정·환경 분석.
2. **학습 평가** (Kirkpatrick L2)
   - 사전·사후 평가 점수 비교.
   - 통과율·평균 점수 산정.
   - 사전·사후 차이 +20% 이상 달성 여부 (PRO §7 KPI).
3. **행동 평가** (Kirkpatrick L3, 선택)
   - 현장 관찰·인터뷰: 학습자가 실무에서 새 스킬을 적용하는지 평가.
   - 표본 추출 방식 (수강생의 10~20%).
4. **결과 평가** (Kirkpatrick L4)
   - 프로젝트 성과 데이터(MA)·OPF 컴플라이언스율 변화 분석.
   - 교육과 성과 지표의 상관 분석.
5. **강사 평가**
   - 강사 평가서 (전문성·전달력·자료품질·소통).
   - 강사 풀 재구성 결정.
6. **개선 제안 도출**
   - 평가 결과에서 커리큘럼·전달방식·자원 측면 개선 제안.
   - OPF SP3.4 교훈 등록부에도 등재.
7. **반기 보고 및 환류**
   - [[TMP-CMMI-01-03-03-01]] 양식 작성.
   - CEO/CTO 보고.
   - 결과 → 차기 사이클 [[WI-CMMI-01-03-01]] SP1.1 입력.

### 5.3 완료 조건
- [ ] 반응·학습 평가 완료 (전 과정 대상)
- [ ] 행동 평가 표본 수행 (해당 과정)
- [ ] 결과 평가 (프로젝트 성과 연동 분석)
- [ ] 강사 평가서 작성·강사 풀 점검
- [ ] 효과성 평가 보고서 작성·승인
- [ ] 개선 제안 등재 + 차기 사이클 입력 인계
- [ ] CM 베이스라인 + MAT-001 등록

## 6. 인터페이스 부서
- **선행 WI**: [[WI-CMMI-01-03-02]] 이수 기록 인수
- **차기 사이클 환류**: [[WI-CMMI-01-03-01]] SP1.1
- **OPF**: 교훈 등록부 연동 ([[WI-CMMI-01-02-04]])
- **MA**: 프로젝트 성과 데이터 공급 ([[PRO-CMMI-04-02]])
- **HR**: 인사 평가·역량 모델 갱신 협조

## 7. 주의사항 / 예외 처리

### 7.1 사전·사후 차이 KPI 미달 (+20% 미달)
- 커리큘럼·교재·강사·평가도구 등 원인 분석.
- 동일 과정 재설계 (3개월 내 갱신 또는 폐지 결정).

### 7.2 행동 평가 표본 부족
- 수강생 10% 미만 표본 시 행동 평가 결론 보류.
- 다음 반기에 표본 확대 또는 인터뷰 추가.

### 7.3 강사 평가 점수 하위
- 강사 풀 평균 대비 하위 10% 강사: 멘토링·재교육 또는 풀 제외.
- 풀 제외 시 [[WI-CMMI-01-03-01]] SP1.4 절차로 대체 확보.
