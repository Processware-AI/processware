# 책임 이전 확인 (WI-CMMI-005-02-08)

> 상위 절차: [[PRO-CMMI-502_공급자_합의_관리_절차_v1.0]]

## 1. 업무 목적
인수 종료 후 공급자→자사 운영 책임 이전을 명확히 확인한다.

## 2. 수행 주체
- 주 수행자: PM
- 검토자: 법무, 운영
- 승인자: Sponsor

## 3. 범위
- 합의 종료 시

## 4. 입력 / 산출물
- Input: 인수 수용서, 합의서
- Output: [[TMP-CMMI-005-02-08_책임_이전_확인서_v1.0]]

## 5. 수행 절차

### 5.1 사전 준비
1. 책임 항목 점검.
2. 자산 인벤토리.

### 5.2 수행 단계
1. **항목 점검**.
2. **자산·계정 회수**.
3. **데이터 인계**.
4. **잔존 의무 정리** — 보증·유지보수.
5. **확인서 서명**.
6. **종결 보고**.

### 5.3 완료 조건
- [ ] 확인서 서명
- [ ] 자산 회수
- [ ] 잔존 의무 등록

## 6. 인터페이스 부서
- 법무, 보안, 운영

## 7. 주의사항 / 예외 처리

### 7.1 자산 누락
- 즉시 회수.

### 7.2 데이터 미인계
- 법적 조치.

### 7.3 보증 기간
- 만료까지 모니터링.
