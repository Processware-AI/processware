# PAL 구축 및 운영 (WI-CMMI-01-02-04)

> 상위 절차: [[PRO-CMMI-01-02_프로세스_자산_개발_절차_v1.0]]

## 1. 업무 목적
조직의 모든 프로세스 자산이 단일 PAL 에서 검색·인용·재사용 가능하도록 인프라·인덱스를 구축·운영하여, 자산의 가시성·접근성을 보장한다.

## 2. 수행 주체
- 주 수행자: SEPG 운영자
- 검토자: IT, PCB
- 승인자: SEPG Lead

## 3. 범위
- vault 디렉토리(03~10), MAT-001 인덱스, 검색 도구

## 4. 입력 / 산출물
- Input: 신규/개정 자산, 폐기 자산
- Output: PAL 등재 결과, [[MAT-001_문서관리대장]] Row, 접근 로그

## 5. 수행 절차

### 5.1 사전 준비
1. 자산 frontmatter 검증(필수 필드).
2. 디렉토리 분류 확인.

### 5.2 수행 단계
1. **등재카드 작성** — 자산 메타데이터·키워드·관련 PA 정리.
2. **인덱스 갱신** — MAT-001 Row 추가/갱신.
3. **검색 키워드 등록** — Obsidian 태그 + frontmatter tags.
4. **접근 권한 설정** — 등급별(공개·내부·기밀) 권한.
5. **공지** — 신규/개정 자산 사내 공지.
6. **접근 통계 모니터링** — 월별 다운로드·조회수 추적.

### 5.3 완료 조건
- [ ] 등재카드 보관
- [ ] MAT-001 Row 등재
- [ ] 검색 시 결과 노출 확인

## 6. 인터페이스 부서
- IT(저장소), 보안팀(접근권한), 커뮤니케이션팀(공지)

## 7. 주의사항 / 예외 처리

### 7.1 기밀 자산
- 기밀 등급 자산은 별도 권한 그룹 운영, 다운로드 시 워터마크 적용.

### 7.2 인덱스 누락
- 분기 점검에서 자산 vs 인덱스 차이 발견 시 30일 내 동기화.

### 7.3 검색 무결과
- 키워드 검색 결과 0건 시 키워드·태그 보강.
