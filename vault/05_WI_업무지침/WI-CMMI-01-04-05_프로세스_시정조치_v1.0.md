# 프로세스 시정조치 (WI-CMMI-01-04-05)

> 상위 절차: [[PRO-CMMI-01-04_구현_인프라_운영_절차_v1.0]]

## 1. 업무 목적
프로세스 부적합·이탈을 시정조치로 발의·추적·종결하여 재발을 막는다.

## 2. 수행 주체
- 주 수행자: 부적합 부서 책임자
- 검토자: QA, SEPG
- 승인자: SEPG Lead

## 3. 범위
- QA 부적합, 내재화 점검 결과, KPI 미달 등

## 4. 입력 / 산출물
- Input: 부적합 보고, KPI 알람
- Output: [[TMP-CMMI-01-04-05-01_프로세스_시정조치서_v1.0]] 등록·종결

## 5. 수행 절차

### 5.1 사전 준비
1. 부적합 분류·중요도 결정.
2. 책임 부서 지정.

### 5.2 수행 단계
1. **시정조치 발의** — 시정조치서 작성.
2. **근본원인 분석** — [[PRO-CMMI-04-03_근본원인분석_및_해결_절차_v1.0]] 적용 가능.
3. **조치 계획** — 즉시·단기·장기.
4. **이행** — 책임 부서 실행.
5. **검증** — QA 가 효과 확인.
6. **종결** — 효과 확인 후 종결.

### 5.3 완료 조건
- [ ] 시정조치서 종결 서명
- [ ] 효과 검증 보고
- [ ] 재발 방지 PAL 등재

## 6. 인터페이스 부서
- QA, SEPG, 부적합 부서

## 7. 주의사항 / 예외 처리

### 7.1 종결 지연
- 30일 초과 시 PCB 보고.

### 7.2 재발
- 동일 원인 재발 시 RCA 의무 + 표준 자산 개정.

### 7.3 책임 회피
- 책임 부서 미이행 시 SEPG Lead 가 직접 지정.
