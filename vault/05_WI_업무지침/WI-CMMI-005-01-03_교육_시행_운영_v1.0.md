# 교육 시행 운영 (WI-CMMI-005-01-03)

> 상위 절차: [[PRO-CMMI-501_조직_훈련_절차_v1.0]]

## 1. 업무 목적
교육 행사를 표준 절차로 운영하여 학습 효과를 높인다.

## 2. 수행 주체
- 주 수행자: OT Lead
- 검토자: HR
- 승인자: HR Director

## 3. 범위
- 모든 교육 행사

## 4. 입력 / 산출물
- Input: 시행 계획
- Output: [[TMP-CMMI-005-01-03_교육_시행_운영표_v1.0]]

## 5. 수행 절차

### 5.1 사전 준비
1. 콘텐츠 점검.
2. 환경(LMS·강의실).

### 5.2 수행 단계
1. **공지**.
2. **수강 등록**.
3. **사전 자료**.
4. **시행**.
5. **만족도 조사**.
6. **수료 처리**.

### 5.3 완료 조건
- [ ] 시행 완료
- [ ] 만족도 조사 회수
- [ ] 이력 등재

## 6. 인터페이스 부서
- HR, IT

## 7. 주의사항 / 예외 처리

### 7.1 환경 장애
- 백업 환경.

### 7.2 강사 결강
- 대체 강사.

### 7.3 자료 유출
- NDA·DRM.
