# 데이터 저장·결과 전달 (WI-CMMI-04-02-03)

> 상위 절차: [[PRO-CMMI-04-02_측정_및_분석_절차]] · 표준: CMMI-DEV V1.3 MA SG2 (후반)

## 1. 업무 목적
수집·분석된 측정 데이터를 측정저장소에 안전하게 저장하고, 분석 결과를 이해관계자에게 적시·정확하게 전달하여 의사결정에 활용되도록 한다. 프로젝트 종료 시 조직 측정저장소(OPD SP1.4)로 통합.

## 2. 수행 주체
- **주 수행자**: Measurement Analyst, Data Owner
- **검토자**: QA Director
- **승인자**: CEO/CTO (조직 차원) 또는 PM (프로젝트 차원)

## 3. 범위
PRO-CMMI-04-02 §5 의 **SP2.3(저장) ~ SP2.4(전달)** 적용. 환류 결과는 [[WI-CMMI-04-02-01]] §5.2.1 의 정보 니즈 재식별로 순환.

## 4. 입력 자료 / 산출물
- **Input**
  - 분석 보고서 ([[WI-CMMI-04-02-02]] 산출)
  - 데이터셋 ([[WI-CMMI-04-02-02]] 산출)
  - 측정저장소 운영규정 ([[WI-CMMI-01-01-03]] 측정저장소 PAL)
- **Output**
  - [[TMP-CMMI-04-02-03-01_측정저장소_등록부]]
  - [[TMP-CMMI-04-02-03-02_측정정보_배포기록]]
  - 조직 측정저장소 신규 등록 (OPD SP1.4 와 연계)

## 5. 수행 절차

### 5.1 사전 준비
1. 측정저장소(PAL) 접근 권한 확인.
2. 보안 분류(공개·내부·기밀·비밀) 검토.
3. 배포 대상자 목록 최신화.

### 5.2 수행 단계
1. **저장 형식 결정 (SP2.3 전반)**
   - 원천 데이터, 가공 데이터, 분석 보고서 각각의 저장 형식·위치 결정.
   - 보관 기간·접근 제어 정의.
2. **측정저장소 등록 (SP2.3 후반)**
   - 데이터셋 메타데이터 + 보고서를 PAL 에 등록.
   - 등록 ID 부여 및 검색 태그 부여.
3. **저장 무결성 점검**
   - 등록 직후 다운로드·해시 검증.
   - 무결성 통과 확인 후 잠금.
4. **이해관계자 배포 (SP2.4 전반)**
   - 결과 배포 양식(보고서 PDF, 대시보드 링크 등) 선택.
   - 보안 분류에 따라 접근 권한 부여.
   - 배포 채널·일시 기록.
5. **피드백 수집·환류**
   - 이해관계자로부터 피드백·후속 정보 니즈 수집.
   - [[WI-CMMI-04-02-01]] §5.2.1 로 환류 입력.
6. **조직 측정저장소 통합 (프로젝트 종료 시)**
   - 프로젝트 측정 데이터를 OPD SP1.4 측정저장소에 통합.
   - 익명화·정규화 필요 시 적용.

### 5.3 완료 조건
- [ ] 모든 데이터셋·보고서가 측정저장소에 등록
- [ ] 접근 권한·보안 분류 적용
- [ ] 등록 직후 무결성 검증 통과
- [ ] 이해관계자 배포 기록 (대상·일시·채널) 보관
- [ ] 피드백 1건 이상 수집 또는 미수집 사유 명기
- [ ] 프로젝트 종료 시 조직 측정저장소 통합 완료

## 6. 인터페이스 부서
- **MA**: 본 지침 주 수행
- **OPD**: 조직 측정저장소 (PAL) 운영자 ([[WI-CMMI-01-01-03]])
- **보안팀**: 분류·접근권한 자문
- **이해관계자**: 결과 수령·피드백
- **PMC/RSKM/QPM**: 측정정보 소비자

## 7. 주의사항 / 예외 처리

### 7.1 보안 분류 위반
- 기밀 데이터를 공개 분류로 잘못 등록한 경우:
  - 즉시 잠금 → 보안팀 통보 → 영향 평가.
  - 외부 노출 시 정보보호 사고 대응 절차로 전환.

### 7.2 배포 후 결과 정정 필요
- 분석 결과 오류 발견 시:
  - 즉시 정정 공지 → 정정본 재배포.
  - 결정에 사용된 의사결정 재검토 권고.

### 7.3 이해관계자 무응답
- 배포 후 일정 기간(2주) 피드백 없으면:
  - 1회 리마인더 발송.
  - 무응답 지속 시 정보 니즈 재검토 ([[WI-CMMI-04-02-01]]).
