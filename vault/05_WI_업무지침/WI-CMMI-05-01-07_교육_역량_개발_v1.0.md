# 교육 역량 개발 (WI-CMMI-05-01-07)

> 상위 절차: [[PRO-CMMI-05-01_조직_훈련_절차_v1.0]]

## 1. 업무 목적
사내 강사·콘텐츠 역량을 지속 개발하여 외부 의존도를 낮춘다.

## 2. 수행 주체
- 주 수행자: OT Lead
- 검토자: HR
- 승인자: HR Director

## 3. 범위
- 사내 강사 풀

## 4. 입력 / 산출물
- Input: 강사 역량 평가
- Output: [[TMP-CMMI-05-01-07-01_강사_역량_개발계획서_v1.0]]

## 5. 수행 절차

### 5.1 사전 준비
1. 강사 풀 점검.
2. 외부 트레이닝.

### 5.2 수행 단계
1. **역량 평가**.
2. **격차**.
3. **개발 계획**.
4. **시행**.
5. **인증**.
6. **모니터링**.

### 5.3 완료 조건
- [ ] 계획서 승인본
- [ ] 인증 통과
- [ ] 강사 풀 갱신

## 6. 인터페이스 부서
- HR, 외부 트레이너

## 7. 주의사항 / 예외 처리

### 7.1 자질 미달
- 코칭 강화.

### 7.2 외부 의존
- 핵심 콘텐츠 사내화.

### 7.3 콘텐츠 노후
- 연 갱신.
