# 프로세스 자원 및 재정 확보 (WI-CMMI-001-01-02)

> 상위 절차: [[PRO-CMMI-101_거버넌스_운영_절차_v1.0]]

## 1. 업무 목적
프로세스 구현과 개선 활동에 필요한 인력·도구·교육·외부 컨설팅 등 자원을 식별·요청·배정·집행하여, 프로세스 활동이 자원 부족으로 중단되지 않도록 보장한다.

## 2. 수행 주체
- **주 수행자**: SEPG Lead, Process Owner
- **검토자**: CFO, PCB
- **승인자**: CEO

## 3. 범위
- 연간 예산 수립 시기(통상 11~12월) + 분기 변경 요청
- 단일 PA 자원이 아닌 전사 프로세스 자원에 한정. 프로젝트 자원은 [[PRO-CMMI-201_프로젝트_계획_절차_v1.0]] 적용.

## 4. 입력 자료 / 산출물
- **Input**: 방향성 선언서, PA별 자원 격차 보고, 전년도 집행 실적
- **Output**: [[TMP-CMMI-001-01-02_프로세스_자원요청서_v1.0]] 승인본, 예산 배정 결과, 집행 모니터링 대시보드

## 5. 수행 절차

### 5.1 사전 준비
1. SEPG 가 PA별 자원 격차(인력·도구·교육)를 표로 수집한다.
2. 전년도 집행 실적·잔액 확인.
3. 우선순위 기준(PA Tier·리스크 등급) 적용.

### 5.2 수행 단계
1. **자원요청서 초안 작성** — Process Owner 가 PA별 항목·금액·기간·근거를 채운다.
2. **SEPG 통합** — 중복 제거·우선순위 정렬·총액 산출.
3. **CFO 재무 검토** — 회사 예산 한도 대비 적정성 검토.
4. **PCB 심의** — 비즈니스 정렬·기대 효과 검증.
5. **CEO 승인** — 최종 배정안 결재.
6. **집행·모니터링** — 월별 집행률 추적, 70% 이하 시 사유 분석.
7. **분기 보고** — PCB 정기회의에 집행 현황 보고.

### 5.3 완료 조건
- [ ] 자원요청서 CEO 서명본 보관
- [ ] 회계시스템에 예산 코드 등록
- [ ] 월별 집행 모니터링 대시보드 운영
- [ ] 분기 PCB 보고 의사록 등재
- [ ] [[MAT-001_문서관리대장]] Row 갱신

## 6. 인터페이스 부서
- **재무팀**: 예산 코드·집행 시스템
- **HR**: 인력 충원
- **구매팀**: 도구·외부 컨설팅 계약
- **PCB**: 심의·모니터링

## 7. 주의사항 / 예외 처리

### 7.1 예산 삭감
- 전사 예산 삭감 시 SEPG 가 우선순위 재정렬, PCB 재심의.

### 7.2 긴급 자원 요청
- 중대 부적합·긴급 사고로 자원이 필요한 경우 CEO 구두 승인 → 30일 내 정식 결재 보완.

### 7.3 미집행 잔액
- 분기 말 미집행 30% 이상 시 사유 분석, 차분기로 이월 또는 회수 결정.
