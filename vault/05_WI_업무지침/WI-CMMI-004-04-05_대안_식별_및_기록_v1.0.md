# 대안 식별 및 기록 (확장) (WI-CMMI-004-04-05)

> 상위 절차: [[PRO-CMMI-404_의사결정_분석_및_해결_절차_v1.0]]

## 1. 업무 목적
관성적 대안 외에 외부·혁신·하이브리드 대안을 적극 식별·기록한다.

## 2. 수행 주체
- 주 수행자: DAR Facilitator
- 검토자: PM, 외부 자문
- 승인자: PM

## 3. 범위
- 모든 DAR

## 4. 입력 / 산출물
- Input: 시장·기술 정보
- Output: [[TMP-CMMI-004-04-05_확장_대안_등록부_v1.0]]

## 5. 수행 절차

### 5.1 사전 준비
1. 외부 자문.
2. 혁신 워크숍.

### 5.2 수행 단계
1. **확장 brainstorming**.
2. **타당성**.
3. **분류**.
4. **기록**.
5. **검토**.
6. **승인**.

### 5.3 완료 조건
- [ ] 등록부
- [ ] 외부 옵션 1개 이상
- [ ] 승인

## 6. 인터페이스 부서
- 외부 자문, R&D

## 7. 주의사항 / 예외 처리

### 7.1 비현실
- 사전 제외 사유 기록.

### 7.2 라이선스
- 외부 옵션 라이선스 점검.

### 7.3 혁신 미흡
- 워크숍 추가.
