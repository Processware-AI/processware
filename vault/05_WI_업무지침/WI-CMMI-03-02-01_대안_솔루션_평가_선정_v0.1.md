# 대안 솔루션 평가·선정 (WI-CMMI-03-02-01)

> 상위 절차: [[PRO-CMMI-03-02_기술_솔루션_설계_절차]]

## 1. 업무 목적
요구사항을 만족하는 대안 솔루션과 평가 기준을 개발하고, 영향 큰 결정에 DAR 을 적용하여 최적 솔루션을 객관적·추적가능하게 선정한다.

## 2. 수행 주체
- **주 수행자**: Chief Engineer / Lead Architect
- **검토자**: DAR Owner, Engineering Director, Domain SME
- **승인자**: Project Manager

## 3. 범위
PRO-CMMI-03-02 §5 의 **SP1.1 (Develop Alternative Solutions and Selection Criteria) ~ SP1.2 (Select Product Component Solutions)** 단계.

## 4. 입력 자료 / 산출물
- **Input**
  - 베이스라인 요구사항 (RD 산출물 / REQM 베이스라인)
  - 기술 트렌드·시장 분석
  - 조직 기술 자산 / 재사용 라이브러리
- **Output**
  - [[TMP-CMMI-03-02-01-01_대안_솔루션_평가_보고서]]
  - [[TMP-CMMI-03-02-01-02_솔루션_선정_결정_보고서]]
  - DAR 결정 기록 (해당 시)

## 5. 수행 절차

### 5.1 사전 준비
1. 베이스라인 요구사항 확보 — RD SP3.5 통과 베이스라인.
2. 영향 큰 결정 식별 — POL-CMMI-04 영향 큰 결정 기준 적용.
3. DAR 적용 여부 결정 — [[PRO-CMMI-04-04_의사결정_분석_결정_절차]] 게이트.

### 5.2 수행 단계
1. **대안 솔루션 후보 도출 (SP1.1)**
   - Make / Buy / Reuse / Hybrid 4축 검토.
   - 후보 최소 3개 이상 — 단일 후보 진행 금지.
2. **선정 기준 정의**
   - 기능 충족 / 성능 / 비용 (CAPEX·OPEX) / 일정 / 리스크 / 기술 성숙도 / 운영성 / 확장성.
   - 각 기준에 가중치 부여 (합 100%).
3. **DAR 적용 (해당 경우)**
   - 영향 큰 결정이면 DAR 형식 평가 의무 — [[PRO-CMMI-04-04]] DAR WI 따름.
   - 비영향 결정도 비교표 작성.
4. **대안별 평가**
   - 각 대안을 기준별 점수화 (1-5 또는 1-10).
   - 가중 합 산출.
5. **민감도 분석**
   - 핵심 기준 가중치 ±20% 변동 시 선정 변동 여부 검토.
   - 변동 발생 시 추가 검증.
6. **솔루션 선정 (SP1.2)**
   - 최고 가중 점수 대안 선정.
   - 동점·근소 차이 시 정성 분석으로 결정.
7. **선정 결정 보고서 작성·결재**
   - 평가·결정 근거·후속 조치를 포함.
   - PM 승인 (DAR 적용 시 DAR 결재 별도).

### 5.3 완료 조건 체크리스트
- [ ] 대안 후보 3개 이상 도출
- [ ] 선정 기준 8축 (또는 프로젝트 정의) 가중치 합 100%
- [ ] 영향 큰 결정 시 DAR 100% 적용
- [ ] 각 대안 기준별 평가표 + 가중 점수 산출
- [ ] 민감도 분석 결과 기록
- [ ] 선정 근거가 평가 결과에 일관
- [ ] PM 승인 결재

## 6. 인터페이스 부서
- **TS 팀**: 주 수행
- **DAR Owner**: 형식 평가 적용 ([[PRO-CMMI-04-04]])
- **Procurement**: Buy 옵션 비용·계약 정보 공급
- **EPG**: Reuse 자산 정보 공급 ([[PRO-CMMI-01-01]])
- **RSKM**: 대안별 리스크 정보 공급 ([[PRO-CMMI-02-06]])

## 7. 주의사항 / 예외 처리

### 7.1 단일 대안만 도출
- 기술적 / 조달적으로 다른 후보가 없는 경우:
  - "단일 대안" 사유 명시 + DAR Owner 검토.
  - 후속 변경 가능성을 PoC / 시범으로 검증 후 본 선정.

### 7.2 평가 기준 불일치
- 이해관계자 간 가중치 합의 불가:
  - 1차 PM 결정, 2차 경영진 격상.
  - 가중치 결정 회의록 첨부.

### 7.3 신규 기술 대안 (성숙도 미검증)
- TRL 4 이하 신규 기술 후보 포함:
  - PoC 의무 + 리스크 등록(RSKM).
  - 본 WI 단계에선 조건부 선정 표기.
