# 기본 모니터링 — 파라미터·약정·리스크·데이터 (WI-CMMI-02-02-01)

> 상위 절차: [[PRO-CMMI-02-02_프로젝트_모니터링_통제_절차]]

## 1. 업무 목적
PP 베이스라인 (일정·예산·노력·범위) 대비 실적을 측정하고, 약정·리스크·데이터관리 활동을 정기적으로 모니터링하여 편차를 조기 식별한다.

## 2. 수행 주체
- **주 수행자**: Project Manager
- **검토자**: PMO Director, Risk Manager (리스크 부분)
- **승인자**: Senior Management (월간 보고)

## 3. 범위
PRO-CMMI-02-02 §5 의 **SP1.1(파라미터) ~ SP1.4(데이터)**.

## 4. 입력 자료 / 산출물
- **Input**
  - PP 베이스라인 (통합 계획서 v1)
  - 약정 서약서 ([[TMP-CMMI-02-01-04-02]])
  - 리스크 목록 ([[TMP-CMMI-02-01-02-02]])
  - 데이터관리 계획서 ([[TMP-CMMI-02-01-02-03]])
  - 주별 실적 데이터 (Jira, 타임시트, EVM 도구)
- **Output**
  - [[TMP-CMMI-02-02-01-01_프로젝트_성과_기록]] (주간)
  - [[TMP-CMMI-02-02-01-02_약정_모니터링_체크리스트]] (월간)
  - 편차 알림 (이슈 분석 큐로 전달)

## 5. 수행 절차

### 5.1 사전 준비
1. 모니터링 주기 확정 (주간 + 월간).
2. 실적 데이터 source 연동 (Jira, EVM 도구 등).
3. 임계치 정의: 일정 ±10%, 비용 ±10%, 리스크 status H 알림.

### 5.2 수행 단계
1. **계획 파라미터 모니터링 (SP1.1)** — 주 1회
   - 일정 진척 (Schedule Variance), 비용 (EVM CV/SV), 노력 (실적/계획), 범위 (완료 WP/전체)
   - 임계치 초과 시 편차 알림 생성
2. **약정 모니터링 (SP1.2)** — 월 1회
   - [[TMP-CMMI-02-02-01-02]] 체크리스트로 각 약정자별 이행 상태 점검
   - 미이행 약정 → 이슈 분석 큐
3. **리스크 모니터링 (SP1.3)** — 주 1회
   - 리스크 목록 status 업데이트 (Active/Closed/Triggered)
   - 신규 리스크 식별 시 [[WI-CMMI-02-06-02]] 입력
4. **데이터관리 모니터링 (SP1.4)** — 월 1회
   - 데이터 분류·보관·접근권한·폐기 정책 준수 점검
   - 위반 발견 시 즉시 시정조치 (보안 사고 발생 시 CISO 통보)
5. **주간/월간 보고서 발행**
   - 주간: 진척·편차 (PM → PMO)
   - 월간: 통합 보고서 (PM → Senior Mgmt)

### 5.3 완료 조건 (주기별 완료 기준)
- [ ] 주간 성과 기록 작성 (매주 금요일까지)
- [ ] 월간 약정 체크리스트 완료
- [ ] 모든 편차 (≥ 임계치) 알림 발행 + 이슈 분석 큐 등록
- [ ] 리스크 status 주간 업데이트 — Triggered 리스크 24h 내 보고
- [ ] 데이터관리 위반 0건 또는 시정조치 등록

## 6. 인터페이스 부서
- **PMO**: 주간 보고 수신
- **Senior Management**: 월간 보고 수신
- **Risk Manager**: 리스크 status 공유 ([[PRO-CMMI-02-06]])
- **CISO/Data Custodian**: 데이터 위반 발생 시 통보
- **MA**: 성과 측정값 PAL 등록 ([[PRO-CMMI-04-02]])

## 7. 주의사항 / 예외 처리

### 7.1 데이터 source 미가용
- Jira/EVM 도구 장애 시 수동 집계 — 다음 정상 cycle 에 재검증.
- 24h 이상 미가용 시 PMO 통보.

### 7.2 임계치 초과 알림 폭주
- 동시 ≥ 5건 편차 발생 시: 단순 알림이 아닌 ESC 회의 소집 (PMO 주관).
- 패턴 발견 시 베이스라인 재검토 (PP 재진입).

### 7.3 약정자 불응 (Non-responsive)
- 약정자 미이행 2주 이상 + 회신 부재: PMO escalation.
- 4주 이상: Sponsor escalation.
