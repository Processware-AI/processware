# 행동제안 구현 및 효과 평가 (WI-CMMI-004-03-04)

> 상위 절차: [[PRO-CMMI-403_근본원인분석_및_해결_절차_v1.0]]

## 1. 업무 목적
RCA 권고 행동을 계획·구현하고 효과를 정량 평가한다.

## 2. 수행 주체
- 주 수행자: RCA 리더, 책임 부서
- 검토자: QA
- 승인자: PM

## 3. 범위
- 모든 RCA 권고

## 4. 입력 / 산출물
- Input: 권고 사항
- Output: [[TMP-CMMI-004-03-04_RCA_행동계획서_v1.0]]

## 5. 수행 절차

### 5.1 사전 준비
1. 자원 확보.
2. KPI 정의.

### 5.2 수행 단계
1. **계획**.
2. **구현**.
3. **모니터링**.
4. **효과 측정**.
5. **종결 또는 재기획**.
6. **교훈**.

### 5.3 완료 조건
- [ ] 행동계획 승인
- [ ] KPI 충족
- [ ] 교훈 등재

## 6. 인터페이스 부서
- SEPG, 모든 팀

## 7. 주의사항 / 예외 처리

### 7.1 효과 미흡
- 재기획.

### 7.2 자원 부족
- PCB 자원 재배정.

### 7.3 부작용
- 대안 검토.
