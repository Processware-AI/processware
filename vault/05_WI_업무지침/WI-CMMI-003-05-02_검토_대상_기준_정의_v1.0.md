# 검토 대상·기준 정의 (WI-CMMI-003-05-02)

> 상위 절차: [[PRO-CMMI-305_동료검토_절차_v1.0]]

## 1. 업무 목적
검토 대상·진입·종료 기준을 명확히 하여 검토 활동의 일관성을 확보한다.

## 2. 수행 주체
- 주 수행자: QA Lead
- 검토자: PM
- 승인자: PM

## 3. 범위
- 모든 산출물 카테고리

## 4. 입력 / 산출물
- Input: 산출물 분류
- Output: [[TMP-CMMI-003-05-02_검토대상_기준서_v1.0]]

## 5. 수행 절차

### 5.1 사전 준비
1. 카테고리 식별.
2. 위험 평가.

### 5.2 수행 단계
1. **대상 결정**.
2. **진입 기준**.
3. **종료 기준**.
4. **검토자 자격**.
5. **승인**.
6. **공유**.

### 5.3 완료 조건
- [ ] 기준서 승인본
- [ ] 검토자 풀 등록
- [ ] 공유

## 6. 인터페이스 부서
- 모든 팀

## 7. 주의사항 / 예외 처리

### 7.1 자격 부족
- 동료 검토자 자격 미달 시 추가 교육.

### 7.2 중복 검토
- 동일 검토자 중복 회피.

### 7.3 외부 검토
- 외부 검토는 라이선스 준수.
