# 공급자 선정기준 운영 (WI-CMMI-005-02-04)

> 상위 절차: [[PRO-CMMI-502_공급자_합의_관리_절차_v1.0]]

## 1. 업무 목적
공급자 선정 시 적용할 표준 기준을 운영한다.

## 2. 수행 주체
- 주 수행자: 구매팀
- 검토자: PCB
- 승인자: Sponsor

## 3. 범위
- 모든 신규 공급자

## 4. 입력 / 산출물
- Input: 인수 범위서
- Output: [[TMP-CMMI-005-02-04_공급자_평가표_v1.0]]

### 5.1 사전 준비
1. 기준 점검.
2. 평가자.

### 5.2 수행 단계
1. **RFP 발송**.
2. **응답 평가**.
3. **현장 점검** — 필요 시.
4. **DAR 적용**.
5. **선정**.
6. **결재**.

### 5.3 완료 조건
- [ ] 평가표 승인
- [ ] 선정 결재
- [ ] 통보
