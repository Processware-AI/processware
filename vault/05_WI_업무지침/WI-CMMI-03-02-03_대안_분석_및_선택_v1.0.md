# 대안 분석 및 선택 (WI-CMMI-03-02-03)

> 상위 절차: [[PRO-CMMI-03-02_기술솔루션_절차_v1.0]]

## 1. 업무 목적
사전 정의된 기준으로 대안을 평가·선택하고 결정 근거를 보존한다.

## 2. 수행 주체
- 주 수행자: 아키텍트
- 검토자: PM
- 승인자: Sponsor

## 3. 범위
- 중대 결정(아키텍처·외부 시스템·주요 도구)

## 4. 입력 / 산출물
- Input: 기준서, 대안 후보
- Output: [[TMP-CMMI-03-02-03-01_대안_분석서_v1.0]], DAR 기록

## 5. 수행 절차

### 5.1 사전 준비
1. 대안 후보 3개 이상.
2. 평가 데이터 확보.

### 5.2 수행 단계
1. **각 대안 평가**.
2. **점수 산출**.
3. **민감도 분석**.
4. **권고 결정**.
5. **Sponsor 승인**.
6. **DAR 적용**(중대 결정).

### 5.3 완료 조건
- [ ] 분석서 승인본
- [ ] 결정 근거 보존
- [ ] DAR 기록

## 6. 인터페이스 부서
- DAR, PMO

## 7. 주의사항 / 예외 처리

### 7.1 단일 대안
- 단일 대안 평가 금지.

### 7.2 점수 동률
- 가중치 재조정 또는 추가 항목.

### 7.3 정치적 결정
- 사후 사유 기록.
