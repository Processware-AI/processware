# 리스크 전략 및 교훈 관리 (WI-CMMI-002-05-08)

> 상위 절차: [[PRO-CMMI-205_리스크_및_기회_관리_절차_v1.0]]

## 1. 업무 목적
조직의 리스크 전략과 사후 교훈을 통합 관리하여 다음 활동의 입력으로 활용한다.

## 2. 수행 주체
- 주 수행자: Risk Manager
- 검토자: PCB
- 승인자: CEO

## 3. 범위
- 연 1회 전략 갱신 + 종결 항목 교훈 등재

## 4. 입력 / 산출물
- Input: 종결 항목, 사고 통계
- Output: [[TMP-CMMI-002-05-08_리스크_전략_및_교훈_v1.0]]

## 5. 수행 절차

### 5.1 사전 준비
1. 종결 항목 수집.
2. 전략 검토 워크숍.

### 5.2 수행 단계
1. **교훈 추출**.
2. **전략 초안**.
3. **PCB 검토**.
4. **CEO 승인**.
5. **공표**.
6. **PAL 등재**.

### 5.3 완료 조건
- [ ] 문서 승인본
- [ ] PAL 등재
- [ ] 교육 자료 갱신

## 6. 인터페이스 부서
- SEPG, OT

## 7. 주의사항 / 예외 처리

### 7.1 비공개 교훈
- 법적 분쟁 등 비공개 부속서.

### 7.2 익명화
- 개인 식별 정보 마스킹.

### 7.3 갱신 누락
- 연 1회 강제, 누락 시 PCB 보고.
