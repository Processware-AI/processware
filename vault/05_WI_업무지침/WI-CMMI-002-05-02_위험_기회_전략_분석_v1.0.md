# 위험·기회 전략 분석 (WI-CMMI-002-05-02)

> 상위 절차: [[PRO-CMMI-205_리스크_및_기회_관리_절차_v1.0]]

## 1. 업무 목적
식별된 위험·기회의 대응 전략(회피·전이·완화·수용·활용)을 분석·결정한다.

## 2. 수행 주체
- 주 수행자: Risk Manager
- 검토자: PMO, Sponsor
- 승인자: Sponsor

## 3. 범위
- High/Medium 등급 항목

## 4. 입력 / 산출물
- Input: 위험 등록부
- Output: [[TMP-CMMI-002-05-02_위험기회_전략분석서_v1.0]]

## 5. 수행 절차

### 5.1 사전 준비
1. 분석 도구(ETA/FTA) 선택.
2. 비용·효과 데이터.

### 5.2 수행 단계
1. **영향 분석**.
2. **전략 후보 도출**.
3. **비용·효과 평가**.
4. **권고 결정**.
5. **Sponsor 승인**.
6. **계획 등록**.

### 5.3 완료 조건
- [ ] 분석서 승인본
- [ ] 전략 결정
- [ ] 대응 계획 등록

## 6. 인터페이스 부서
- 법무, 보험, 보안

## 7. 주의사항 / 예외 처리

### 7.1 비용 과다
- 수용 가능 임계값 초과 시 보류·재분석.

### 7.2 윤리·법규
- 회피·전이 시 윤리·법적 검토.

### 7.3 기회 우선순위
- 기회는 위험과 동일 가중치로 비교.
