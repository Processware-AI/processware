# 데이터 획득 및 분석 (WI-CMMI-002-04-04)

> 상위 절차: [[PRO-CMMI-204_성과_및_측정_관리_절차_v1.0]]

## 1. 업무 목적
수집된 측정 데이터를 통계적으로 분석·해석하여 의사결정에 활용한다.

## 2. 수행 주체
- 주 수행자: MPM 분석가
- 검토자: PCB
- 승인자: MPM Lead

## 3. 범위
- 분기·연간 분석

## 4. 입력 / 산출물
- Input: 측정저장소
- Output: [[TMP-CMMI-002-04-04_분석_리포트_v1.0]]

## 5. 수행 절차

### 5.1 사전 준비
1. 분석 가설 정의.
2. 데이터 정합성 점검.

### 5.2 수행 단계
1. **추출**.
2. **전처리**.
3. **기술통계**.
4. **추론통계**(필요 시).
5. **시각화**.
6. **해석·권고**.

### 5.3 완료 조건
- [ ] 리포트 작성
- [ ] 권고 도출
- [ ] PCB 보고

## 6. 인터페이스 부서
- IT, 각 부서

## 7. 주의사항 / 예외 처리

### 7.1 표본 부족
- N<30 시 정성 보조.

### 7.2 외부 변수
- 분리 분석.

### 7.3 해석 편향
- 2인 검토 의무.
