# 통합 환경 관리 (WI-CMMI-03-03-03)

> 상위 절차: [[PRO-CMMI-03-03_제품통합_절차_v1.0]]

## 1. 업무 목적
통합 환경의 일관성·격리성을 유지하여 환경 차이로 인한 결함을 차단한다.

## 2. 수행 주체
- 주 수행자: DevOps
- 검토자: Integration Lead
- 승인자: PM

## 3. 범위
- DEV/STG/PROD-mirror 환경

## 4. 입력 / 산출물
- Input: 환경 사양
- Output: [[TMP-CMMI-03-03-03-01_통합환경_정의서_v1.0]], 환경 이미지

## 5. 수행 절차

### 5.1 사전 준비
1. 환경 정의서 갱신.
2. 자동화 스크립트.

### 5.2 수행 단계
1. **셋업**.
2. **데이터 준비**.
3. **격리 검증**.
4. **상태 모니터링**.
5. **롤백 절차**.
6. **회수**.

### 5.3 완료 조건
- [ ] 환경 가용성 ≥ 99%
- [ ] 격리 검증 통과
- [ ] 롤백 가능 확인

## 6. 인터페이스 부서
- IT, 보안

## 7. 주의사항 / 예외 처리

### 7.1 데이터 누설
- 운영 데이터 익명화.

### 7.2 환경 표류
- 자동 재구축 주기 적용.

### 7.3 비용
- 자동 정지 정책.
