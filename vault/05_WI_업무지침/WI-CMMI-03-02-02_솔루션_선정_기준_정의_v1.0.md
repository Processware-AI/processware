# 솔루션 선정 기준 정의 (WI-CMMI-03-02-02)

> 상위 절차: [[PRO-CMMI-03-02_기술솔루션_절차_v1.0]]

## 1. 업무 목적
대안 평가에 사용할 기준·가중치를 사전에 정의하여 평가의 객관성을 확보한다.

## 2. 수행 주체
- 주 수행자: 아키텍트
- 검토자: PM
- 승인자: PM

## 3. 범위
- 중대 솔루션 결정 직전

## 4. 입력 / 산출물
- Input: 요구·제약·전략
- Output: [[TMP-CMMI-03-02-02-01_솔루션_선정기준서_v1.0]]

## 5. 수행 절차

### 5.1 사전 준비
1. 평가 항목 후보.
2. 가중치 합 100%.

### 5.2 수행 단계
1. **항목 도출**.
2. **가중치 결정**.
3. **검토**.
4. **승인**.
5. **DAR 등록**.
6. **버전 관리**.

### 5.3 완료 조건
- [ ] 기준서 승인본
- [ ] 가중치 합 100%
- [ ] 평가 시 인용

## 6. 인터페이스 부서
- DAR, PMO

## 7. 주의사항 / 예외 처리

### 7.1 가중치 편향
- 다중 검토자.

### 7.2 외부 강제
- 강제 시 사유 기록.

### 7.3 변경
- 평가 직전 변경 금지.
