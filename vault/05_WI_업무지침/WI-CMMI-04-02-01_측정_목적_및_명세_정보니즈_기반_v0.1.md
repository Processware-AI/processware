# 측정 목적·명세 — 정보 니즈 기반 (WI-CMMI-04-02-01)

> 상위 절차: [[PRO-CMMI-04-02_측정_및_분석_절차]] · 표준: CMMI-DEV V1.3 MA SG1

## 1. 업무 목적
이해관계자의 정보 니즈를 식별·분석하여 측정 목적·명세(base + derived)·수집·분석 절차를 정의함으로써 의사결정 가능한 측정정보 제공의 토대를 구축한다.

## 2. 수행 주체
- **주 수행자**: Measurement Analyst
- **검토자**: 이해관계자(경영진/PM/QA), Data Owner
- **승인자**: CEO/CTO (조직 측정 체계 차원) 또는 PM (프로젝트 차원)

## 3. 범위
PRO-CMMI-04-02 §5 의 **SP1.1(측정 목적) ~ SP1.4(분석 명세)** 적용. 데이터 수집·분석 실행은 [[WI-CMMI-04-02-02]] 참조.

## 4. 입력 자료 / 산출물
- **Input**
  - 비즈니스 목표·이해관계자 요구
  - 정보 니즈 인터뷰·워크숍 결과
  - 기존 측정 자산(OPD SP1.4 측정 저장소)
- **Output**
  - [[TMP-CMMI-04-02-01-01_측정목적_정의서]]
  - [[TMP-CMMI-04-02-01-02_측정_명세서]] (base + derived)
  - [[TMP-CMMI-04-02-01-03_수집_저장_분석_절차서]]

## 5. 수행 절차

### 5.1 사전 준비
1. 이해관계자 식별 (경영진·PM·QA·고객 등).
2. 기존 측정 저장소(OPD SP1.4) 자산 조회 — 재사용 후보 파악.
3. 정보 니즈 도출 워크숍 일정 확정.

### 5.2 수행 단계
1. **정보 니즈 식별·우선순위 (SP1.1 전반)**
   - 이해관계자별 의사결정 시나리오 인터뷰.
   - 정보 니즈 후보를 비즈니스 목표·전략 부합도로 우선순위화.
2. **측정 목적 수립 (SP1.1 후반)**
   - 각 정보 니즈에 대응하는 측정 목적을 문서화.
   - SMART(특정·측정가능·달성가능·관련성·시한) 점검.
3. **측정 명세 (SP1.2)**
   - Base measure: 직접 측정값 정의 (수집 단위·정밀도·빈도).
   - Derived measure: base 의 함수(비율·합·평균) 정의.
   - 측정 단위·임계값·해석 가이드 명시.
4. **수집·저장 절차 (SP1.3)**
   - 데이터 수집 방법(자동/수동), 출처(시스템·툴), 책임자 정의.
   - 저장소·스키마·보관 기간 결정.
   - 무결성 점검 룰(중복·범위·타입) 정의.
5. **분석 명세 (SP1.4)**
   - 분석 절차(통계·추세·비교·예측) 명시.
   - 결과 표현 양식(대시보드·표·그래프) 정의.
   - 보고 주기·대상 정의.
6. **검토·승인**
   - 이해관계자 검토 → 명세 확정 → 결재.

### 5.3 완료 조건
- [ ] 정보 니즈 목록 우선순위 부여 완료
- [ ] 측정 목적이 SMART 점검 통과
- [ ] 측정 명세서 (base+derived) 작성·검토 완료
- [ ] 수집·저장·분석 절차 명시
- [ ] 이해관계자 검토 의견 반영
- [ ] 결재 완료 및 MAT-001 등록

## 6. 인터페이스 부서
- **MA**: 본 지침 주 수행
- **OPD**: 측정 저장소(PAL) 연계 ([[WI-CMMI-01-01-03]])
- **이해관계자**: 정보 니즈 공급
- **Data Owner**: 데이터 수집 책임
- **All PAs**: MA 는 전 PA 의 측정 지원 (MA-supports-all)

## 7. 주의사항 / 예외 처리

### 7.1 정보 니즈 충돌
- 이해관계자 간 정보 니즈가 상충 (예: 경영진은 종합 KPI / 개발자는 상세 메트릭) 시:
  - 측정 명세를 계층화 (요약 + 상세).
  - 상위 측정 우선 — 상세는 옵션으로.

### 7.2 측정 가능성 부족
- 정보 니즈가 직접 측정 어려운 경우(예: "팀 사기"):
  - Proxy 지표(이직률·설문 점수)로 대체.
  - 한계 명시: 측정 명세에 "proxy" 표기.

### 7.3 측정 비용 과다
- 측정 1건당 인일 ≥ 1 인 경우 자동화 검토 필수.
- 자동화 불가 시 표본 측정(전수 대비) 검토.
