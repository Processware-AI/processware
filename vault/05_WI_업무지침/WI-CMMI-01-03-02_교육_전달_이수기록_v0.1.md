# 교육 전달·이수 기록 (WI-CMMI-01-03-02)

> 상위 절차: [[PRO-CMMI-01-03_조직_훈련_절차]]

## 1. 업무 목적
전술계획에 따라 교육을 실제 전달하고, 출석·이수·waiver(면제)를 빠짐없이 기록·관리하여, 인력 역량 증빙 + 효과성 평가의 기초 데이터를 확보한다.

## 2. 수행 주체
- **주 수행자**: Training Manager (운영), 강사
- **검토자**: Process Owner (해당 PA 과정 시)
- **승인자**: Training Manager (이수 기록 확정)

## 3. 범위
PRO-CMMI-01-03 §5 의 **SP2.1(교육 전달) ~ SP2.2(이수 기록)** 단계. 효과성 평가는 [[WI-CMMI-01-03-03]] 참조.

## 4. 입력 자료 / 산출물
- **Input**
  - 승인된 전술계획 + 커리큘럼·강사 풀 ([[WI-CMMI-01-03-01]] 산출)
  - 대상자 명단 (역할 매트릭스 기반)
  - LMS·교육실 등 자원
- **Output**
  - [[TMP-CMMI-01-03-02-01_교육_세션_운영기록]] 작성본 (회차별)
  - [[TMP-CMMI-01-03-02-02_교육_이수_기록부]] 갱신본
  - 강사 평가·만족도 설문 raw data
  - waiver 결재 문서

## 5. 수행 절차

### 5.1 사전 준비
1. 회차별 대상자 확정 — HR·PM 협조.
2. 강사 배정·교재 검수.
3. LMS 등록·알림 발송.
4. waiver 신청 사전 접수(공지 시점부터).

### 5.2 수행 단계
1. **대상자 선정 및 통보** (SP2.1)
   - 역할 매트릭스 기반 대상자 식별.
   - 사전 공지 (최소 2주 전, 긴급 시 3일 전).
   - 출석·면제 의사 확인.
2. **자원 일정 확정** (SP2.1)
   - 강사·교육실·LMS 슬롯 예약.
   - 교재·실습장비 준비 점검.
3. **교육 전달** (SP2.1)
   - 강사 진행 + 운영자 출석 체크.
   - [[TMP-CMMI-01-03-02-01]] 양식에 회차별 기록.
   - 사후 만족도 설문(4점 척도 + 코멘트).
4. **전달 추적** (SP2.1)
   - 전술계획 대비 이수 인원·진행률 주간 추적.
   - 미달 시 보강 회차 편성.
5. **이수 기록 작성** (SP2.2)
   - 출석·평가 결과 → [[TMP-CMMI-01-03-02-02]] 등재.
   - 통과·불통과 명시 (평가 기준 점수 적용).
   - 불통과자에 대한 재교육·재시험 일정 통보.
6. **waiver 처리** (SP2.2)
   - waiver 신청(증빙: 외부 자격증·이전 이수 등) → 검토 → 승인.
   - waiver 사유와 함께 기록부에 등재.
7. **기록 보관 및 가용성 제공** (SP2.2)
   - LMS·종이 결재본 보관.
   - 인사 배치·평가 시 조회 가능하도록 권한 부여.
   - 보관기간: 7년 (또는 회사 인사규정에 따름).

### 5.3 완료 조건
- [ ] 회차별 운영기록 작성 (모든 회차)
- [ ] 이수 기록부 등재 — 출석·평가·통과 명시
- [ ] 불통과자 재교육 일정 통지
- [ ] waiver 결재 문서 보관
- [ ] 만족도 설문 결과 raw data 수집·저장
- [ ] LMS/EDMS 권한 매트릭스 적용
- [ ] 이수율·만족도 KPI 집계

## 6. 인터페이스 부서
- **선행 WI**: [[WI-CMMI-01-03-01]] 전술계획·커리큘럼 인수
- **후행 WI**: [[WI-CMMI-01-03-03]] 효과성 평가 (기록 데이터 공급)
- **HR**: 대상자 명단·인사 배치 연동
- **PM/부서장**: 대상자 출석 협조
- **IT**: LMS 운영 지원

## 7. 주의사항 / 예외 처리

### 7.1 출석률 저조 (이수율 95% 미달)
- 강사·과정·일정·예고 등 원인 분석 → 재계획.
- 부서장에게 출석 협조 공식 요청.

### 7.2 강사 결강·취소
- 대체 강사 풀 우선 활용.
- 불가 시 재일정 또는 이러닝 대체.
- 수강생에게 즉시 공지.

### 7.3 waiver 남용
- 같은 학습자 동일 PA 영역 waiver 2회 이상 발견 시 EPG 와 협의.
- 면제 기준 강화 또는 재평가 의무화.
