# 부적합 식별 및 기록 (WI-CMMI-04-01-01)

> 상위 절차: [[PRO-CMMI-04-01_프로세스_품질보증_절차_v1.0]]

## 1. 업무 목적
QA 활동에서 발견된 부적합을 표준 양식으로 기록하여 시정 추적의 입력으로 사용한다.

## 2. 수행 주체
- 주 수행자: QA
- 검토자: PM
- 승인자: PM

## 3. 범위
- 모든 QA 평가 결과

## 4. 입력 / 산출물
- Input: 평가 결과
- Output: [[TMP-CMMI-04-01-01-01_부적합_등록서_v1.0]]

## 5. 수행 절차

### 5.1 사전 준비
1. 분류 기준.
2. 등록부 접근.

### 5.2 수행 단계
1. **식별**.
2. **분류** — Critical/Major/Minor.
3. **즉시 통보** — Critical 24h.
4. **등록**.
5. **책임자 지정**.
6. **시정 발의**.

### 5.3 완료 조건
- [ ] 등록서 보관
- [ ] 책임자 통보
- [ ] 시정 발의

## 6. 인터페이스 부서
- 모든 팀

## 7. 주의사항 / 예외 처리

### 7.1 식별 누락
- 사후 발견 시에도 등록.

### 7.2 부서 회피
- 회피 시 PCB 보고.

### 7.3 보안 부적합
- 별도 보안 채널.
