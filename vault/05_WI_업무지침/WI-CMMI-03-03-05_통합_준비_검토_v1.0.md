# 통합 준비 검토 (WI-CMMI-03-03-05)

> 상위 절차: [[PRO-CMMI-03-03_제품통합_절차_v1.0]]

## 1. 업무 목적
통합 직전 컴포넌트·환경·인력의 준비 상태를 점검하여 통합 실패를 사전 차단한다.

## 2. 수행 주체
- 주 수행자: Integration Lead
- 검토자: QA, 개발 리드
- 승인자: PM

## 3. 범위
- 통합 직전

## 4. 입력 / 산출물
- Input: 컴포넌트 산출물, 환경
- Output: [[TMP-CMMI-03-03-05-01_통합_준비_검토표_v1.0]]

## 5. 수행 절차

### 5.1 사전 준비
1. 검토 항목 점검.
2. 인력 일정.

### 5.2 수행 단계
1. **컴포넌트 점검**.
2. **환경 점검**.
3. **인터페이스 검토**.
4. **인력 점검**.
5. **Go/No-Go 결정**.
6. **승인**.

### 5.3 완료 조건
- [ ] 검토표 모든 ✅
- [ ] Go 결정 또는 시정
- [ ] 결정 회의록

## 6. 인터페이스 부서
- 개발, QA

## 7. 주의사항 / 예외 처리

### 7.1 No-Go
- 시정 후 재검토.

### 7.2 부분 Go
- 범위 축소 통합.

### 7.3 외부 미준비
- 일정 조정.
