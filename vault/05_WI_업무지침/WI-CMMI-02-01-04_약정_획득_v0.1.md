# 약정 획득 — 검토·조정·약정 (WI-CMMI-02-01-04)

> 상위 절차: [[PRO-CMMI-02-01_프로젝트_계획_절차]]

## 1. 업무 목적
WI-03 산출 통합 계획서를 상위·인접 이해관계자와 검토하고, 자원·일정·범위 조정 후 모든 관련자의 약정을 서면 획득하여 프로젝트 베이스라인을 확정한다.

## 2. 수행 주체
- **주 수행자**: Project Manager
- **검토자**: PMO Director, 인접 PA 책임자
- **승인자**: CEO/CTO (Sponsor)

## 3. 범위
PRO-CMMI-02-01 §5 의 **SP3.1(검토) ~ SP3.3(약정)**. 본 WI 종료 시 PMC 절차 ([[PRO-CMMI-02-02]]) 로 인계.

## 4. 입력 자료 / 산출물
- **Input**
  - [[TMP-CMMI-02-01-03-03]] 통합 프로젝트 계획서 v0
  - 상위·인접 PA 책임자 명단
  - Sponsor 약정 가능 시간·예산 한도
- **Output**
  - [[TMP-CMMI-02-01-04-01_약정_검토_조정_기록]]
  - [[TMP-CMMI-02-01-04-02_약정_서약서]]
  - 베이스라인화된 통합 계획서 v1 (CM 등록)

## 5. 수행 절차

### 5.1 사전 준비
1. 통합 계획서 v0 + 부속 문서 인쇄/PDF.
2. 약정 회의 참석자 confirm.
3. 회의 의제 송부 (3일 전).

### 5.2 수행 단계
1. **상위·인접 계획 검토 (SP3.1)**
   - Sponsor, PMO, 인접 PA 책임자 검토 회의 (2시간 권장)
   - 자원 충돌·일정 충돌·예산 cap 등 점검
   - 검토 의견 [[TMP-CMMI-02-01-04-01]] §검토의견 에 기재
2. **자원·작업 조정 (SP3.2)**
   - 검토 의견 기반 일정·예산·범위 조정
   - 조정 안 부결 시 sponsor escalation
   - 조정 결과 [[TMP-CMMI-02-01-04-01]] §조정사항 에 기재
3. **약정 획득 (SP3.3)**
   - 조정 완료 통합 계획서를 최종본으로 확정
   - [[TMP-CMMI-02-01-04-02]] 약정 서약서에 모든 약정자 서명·날인
   - 약정 검토 결과 보고서 (sponsor 결재) 첨부
4. **베이스라인화·CM 등록**
   - [[PRO-CMMI-04-01]] 형상관리 절차로 CM 등록
   - 통합 계획서 v0 → v1 로 버전 증가
   - [[MAT-001]] 등록
5. **PMC 인계**
   - PMC 절차에 베이스라인 ID·약정 서약서 인계

### 5.3 완료 조건
- [ ] 상위·인접 검토 의견 전건 처리 (해소 또는 escalate)
- [ ] 자원·작업 조정 결과 통합 계획서 반영
- [ ] 약정 서약서에 모든 약정자 서명·날인
- [ ] Sponsor 결재 첨부
- [ ] CM 베이스라인 ID 부여 + MAT-001 등록
- [ ] PMC 인계 완료 (인계 ID 발급)

## 6. 인터페이스 부서
- **Sponsor (CEO/CTO)**: 최종 약정 결재
- **PMO**: 검토 의견 종합·이슈 조정
- **인접 PA 책임자 (RD/CM/MA/OT/QA)**: 자기 PA 영향 사항 검토·약정
- **PMC**: 베이스라인 인계 ([[PRO-CMMI-02-02]] §5 진입)

## 7. 주의사항 / 예외 처리

### 7.1 약정자 불참 또는 거부
- 불참자 1명 발생 시 회의 연기 (2주 내).
- 명시적 거부 시 sponsor escalation + 거부 사유 기록.
- 거부 사유가 PA 책임자의 자원 부족인 경우 → 자원 조정 또는 일정 조정 분기.

### 7.2 베이스라인 후 즉시 변경 요청
- 약정 후 1주 내 변경 요청 발생 시: WI-01~03 재실행이 아닌 PMC SP2.2 (시정조치) 로 처리.
- 1주 이후 30% 이상 영향 변경 요청은 PP 재진입 (메이저 베이스라인 변경).

### 7.3 Sponsor 예산 cap 초과
- 통합 계획서 총예산 > Sponsor cap 인 경우: 범위 축소 우선, 일정 연장 차순.
- 최후 수단으로 sponsor 와 cap 재협의.
