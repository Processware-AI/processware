# 규모 추정 (WI-CMMI-02-03-02)

> 상위 절차: [[PRO-CMMI-02-03_추정_관리_절차_v1.0]]

## 1. 업무 목적
프로젝트 산출물 규모를 정량 추정하여 공수·일정·비용 추정의 기준으로 삼는다.

## 2. 수행 주체
- 주 수행자: EST 전문가
- 검토자: PMO
- 승인자: PM

## 3. 범위
- 신규 프로젝트, 요구 변경 시

## 4. 입력 / 산출물
- Input: 요구사항, 아키텍처
- Output: [[TMP-CMMI-02-03-02-01_규모_추정서_v1.0]]

## 5. 수행 절차

### 5.1 사전 준비
1. 추정 단위 결정(FP/LoC/스토리포인트).
2. 카운팅 규칙 확인.

### 5.2 수행 단계
1. **요구 분해**.
2. **단위별 카운트**.
3. **복잡도 가중**.
4. **소계·총계 산출**.
5. **검증** — 동료 검토.
6. **저장소 적재**.

### 5.3 완료 조건
- [ ] 추정서 승인본
- [ ] 검토 의견 반영
- [ ] 측정저장소 적재

## 6. 인터페이스 부서
- PMO, MPM

## 7. 주의사항 / 예외 처리

### 7.1 단위 혼용 금지
- 한 프로젝트는 단일 단위.

### 7.2 변경
- 요구 변경 시 규모 재산정.

### 7.3 외부 추정
- 외부 컨설턴트 추정 결과는 내부 검증 필수.
