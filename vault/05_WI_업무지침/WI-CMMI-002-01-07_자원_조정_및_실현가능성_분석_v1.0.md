# 자원 조정 및 실현가능성 분석 (WI-CMMI-002-01-07)

> 상위 절차: [[PRO-CMMI-201_프로젝트_계획_절차_v1.0]]

## 1. 업무 목적
계획된 범위·일정·예산이 가용 자원으로 실현 가능한지 분석하고 격차를 조정하여 실패 위험을 사전 차단한다.

## 2. 수행 주체
- 주 수행자: PM
- 검토자: PMO
- 승인자: Sponsor

## 3. 범위
- 베이스라인 직전, 단계 게이트

## 4. 입력 / 산출물
- Input: 추정·예산·일정, 자원 풀
- Output: [[TMP-CMMI-002-01-07_실현가능성_분석서_v1.0]] 승인본

## 5. 수행 절차

### 5.1 사전 준비
1. 가용 자원 풀 확인.
2. 외부 의존성 식별.

### 5.2 수행 단계
1. **격차 식별** — 요구 vs 가용.
2. **조정 옵션** — 범위·일정·인력 조정.
3. **시나리오 분석** — 보수·낙관·비관.
4. **리스크 등록**.
5. **권고 결정** — Sponsor 결정.
6. **계획 갱신**.

### 5.3 완료 조건
- [ ] 분석서 승인본
- [ ] 시나리오 3종 비교
- [ ] 결정 회의록

## 6. 인터페이스 부서
- HR, 재무팀, 부서장

## 7. 주의사항 / 예외 처리

### 7.1 실현 불가
- 실현 불가 판단 시 프로젝트 보류·재기획.

### 7.2 외부 의존
- 외부 자원 의존 시 계약 사전 점검.

### 7.3 정치적 압력
- 정치적 압력으로 강제 추진 시 리스크 등록부에 명시.
