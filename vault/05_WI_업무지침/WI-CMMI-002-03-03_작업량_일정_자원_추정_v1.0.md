# 작업량·일정·자원 추정 (WI-CMMI-002-03-03)

> 상위 절차: [[PRO-CMMI-203_추정_관리_절차_v1.0]]

## 1. 업무 목적
규모 추정과 생산성 데이터를 결합해 공수·일정·자원을 추정한다.

## 2. 수행 주체
- 주 수행자: PM
- 검토자: EST 전문가
- 승인자: PM

## 3. 범위
- 모든 신규 프로젝트, 단계 게이트

## 4. 입력 / 산출물
- Input: 규모 추정, 생산성 baseline
- Output: [[TMP-CMMI-002-03-03_공수_일정_자원_추정서_v1.0]]

## 5. 수행 절차

### 5.1 사전 준비
1. 생산성 데이터 확보.
2. 자원 풀 확인.

### 5.2 수행 단계
1. **공수 산출**.
2. **일정 환산**.
3. **자원 매핑**.
4. **버퍼·리스크 충당**.
5. **타당성 검증**.
6. **저장소 적재**.

### 5.3 완료 조건
- [ ] 추정서 승인본
- [ ] 자원 가용성 검증
- [ ] 저장소 적재

## 6. 인터페이스 부서
- HR, 부서장

## 7. 주의사항 / 예외 처리

### 7.1 자원 부족
- 부족 시 외부 인력 또는 일정 조정.

### 7.2 생산성 차이
- 팀 신규성·도메인 차이로 생산성 변동 시 조정 계수 적용.

### 7.3 일정 압박
- 압박 시 리스크 등록.
