# 대응계획 수립 (WI-CMMI-02-05-05)

> 상위 절차: [[PRO-CMMI-02-05_리스크_및_기회_관리_절차_v1.0]]

## 1. 업무 목적
위험·기회 항목별 대응 활동·트리거·자원·책임을 명문화한다.

## 2. 수행 주체
- 주 수행자: Risk Manager
- 검토자: PMO
- 승인자: Sponsor

## 3. 범위
- High 항목 필수, Medium 권고

## 4. 입력 / 산출물
- Input: 전략 분석 결과
- Output: [[TMP-CMMI-02-05-05-01_위험기회_대응계획서_v1.0]]

## 5. 수행 절차

### 5.1 사전 준비
1. 자원 가용성 확인.
2. 트리거 지표 정의.

### 5.2 수행 단계
1. **활동 정의**.
2. **트리거 명시**.
3. **자원 배정**.
4. **책임 지정**.
5. **승인**.
6. **등록**.

### 5.3 완료 조건
- [ ] 계획서 승인본
- [ ] 트리거 모니터링 셋업
- [ ] 자원 배정 확인

## 6. 인터페이스 부서
- 자원 보유 부서

## 7. 주의사항 / 예외 처리

### 7.1 자원 부족
- 부족 시 재협상 또는 전략 변경.

### 7.2 트리거 모호
- 정량 지표로 재정의.

### 7.3 다중 위험
- 의존 위험 통합 대응.
