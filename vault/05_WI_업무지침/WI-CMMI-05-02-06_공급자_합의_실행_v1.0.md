# 공급자 합의 실행 (WI-CMMI-05-02-06)

> 상위 절차: [[PRO-CMMI-05-02_공급자_합의_관리_절차_v1.0]]

## 1. 업무 목적
공급자가 합의대로 이행하는지 모니터링하고 부적합을 시정한다.

## 2. 수행 주체
- 주 수행자: PM
- 검토자: QA
- 승인자: PM

## 3. 범위
- 진행 중 합의

## 4. 입력 / 산출물
- Input: 합의서, 이행 데이터
- Output: [[TMP-CMMI-05-02-06-01_공급자_이행_점검표_v1.0]]

## 5. 수행 절차

### 5.1 사전 준비
1. 모니터링 지표.
2. 정기 미팅.

### 5.2 수행 단계
1. **마일스톤 점검**.
2. **품질 점검**.
3. **결제 처리**.
4. **부적합 시정**.
5. **에스컬레이션**.
6. **종결 검증**.

### 5.3 완료 조건
- [ ] 점검표 갱신
- [ ] 부적합 종결
- [ ] 결제

## 6. 인터페이스 부서
- 재무, 법무

## 7. 주의사항 / 예외 처리

### 7.1 위반
- 페널티 발동.

### 7.2 의사소통 부재
- 정기 미팅 의무.

### 7.3 보안 사고
- 즉시 차단.
