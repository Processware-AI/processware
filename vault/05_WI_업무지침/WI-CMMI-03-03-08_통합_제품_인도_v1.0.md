# 통합 제품 인도 (WI-CMMI-03-03-08)

> 상위 절차: [[PRO-CMMI-03-03_제품통합_절차_v1.0]]

## 1. 업무 목적
통합 완료 제품을 운영·고객에 안전하게 인도하고 인수 서명을 받는다.

## 2. 수행 주체
- 주 수행자: PM
- 검토자: QA, 운영
- 승인자: Sponsor

## 3. 범위
- 릴리스·운영 인계

## 4. 입력 / 산출물
- Input: 통합 결과
- Output: [[TMP-CMMI-03-03-08-01_제품_인도서_v1.0]]

## 5. 수행 절차

### 5.1 사전 준비
1. 인수 기준 합의.
2. 매뉴얼·트레이닝.

### 5.2 수행 단계
1. **릴리스 빌드 생성**.
2. **인수 검증**.
3. **운영 배포**.
4. **모니터링**.
5. **인수 서명**.
6. **공식 종결**.

- [ ] 인도서 서명
- [ ] hypercare 계획
- [ ] 베이스라인 등록
