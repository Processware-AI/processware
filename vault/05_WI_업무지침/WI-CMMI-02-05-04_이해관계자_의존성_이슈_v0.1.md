# 이해관계자 협업·의존성·이슈 해결 (WI-CMMI-02-05-04)

> 상위 절차: [[PRO-CMMI-02-05_통합_프로젝트_관리_절차]]

## 1. 업무 목적
이해관계자 협업 회의를 운영하고, 외부·내부 의존성을 추적하며, 협업·의존성에서 발생하는 이슈를 해결한다.

## 2. 수행 주체
- **주 수행자**: Project Manager
- **검토자**: Stakeholder Manager
- **승인자**: PMO Director

## 3. 범위
PRO-CMMI-02-05 §5 의 **SG2 SP2.1~2.3**.

## 4. 입력 자료 / 산출물
- **Input**: 이해관계자 참여 계획서 (PP WI-03), 통합 계획서, 의존성 (외부 공급자, 내부 인접 PA)
- **Output**: 협업 회의록, 의존성 대장, 이슈 해결 기록

## 5. 수행 절차

### 5.1 사전 준비
1. 참여 계획서 + 의존성 초기 식별.

### 5.2 수행 단계
1. **이해관계자 협업 (SP2.1)** — 정기 회의 운영 (계획서 빈도 따름), 회의록 작성.
2. **의존성 관리 (SP2.2)** — 외부 (공급자·고객)·내부 (인접 PA) 의존성 목록 관리, 상태 추적.
3. **이슈 해결 (SP2.3)** — 협업·의존성에서 발생한 이슈 분석·해결.

### 5.3 완료 조건
- [ ] 정기 협업 회의 (계획서 빈도 준수)
- [ ] 의존성 대장 — 모든 의존성 추적
- [ ] Critical 의존성 — 위반 0건 또는 mitigation 등록
- [ ] 이슈 해결 기록 — 모든 발생 이슈 처리

## 6. 인터페이스 부서
- **각 이해관계자**: 회의 참여
- **공급자**: 외부 의존성 ([[PRO-CMMI-02-04]] 연계)
- **PMC**: 이슈 해결 시정조치 큐 ([[WI-CMMI-02-02-03]])

## 7. 주의사항 / 예외 처리

### 7.1 회의 불참 빈발
- 동일 이해관계자 3회 연속 불참 시 1:1 면담.
- 이해관계자 참여 계획 재수립 ([[WI-CMMI-02-01-03]] 재실행).

### 7.2 Critical 의존성 위반
- 외부 공급자 의존성 위반 시 SAM 협약 §벌칙 발동.
- 내부 PA 의존성 위반 시 PMC SP2 시정조치 + 해당 PA 책임자 escalation.

### 7.3 이슈 해결 교착
- 협업 이슈 해결 안 될 시 Sponsor 의견 청취 + PMO 결정.
