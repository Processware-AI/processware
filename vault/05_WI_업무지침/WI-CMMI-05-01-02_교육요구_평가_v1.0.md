# 교육요구 평가 (WI-CMMI-05-01-02)

> 상위 절차: [[PRO-CMMI-05-01_조직_훈련_절차_v1.0]]

## 1. 업무 목적
직무·전략 기반 교육 요구를 정기 평가하여 격차 해소 입력으로 활용한다.

## 2. 수행 주체
- 주 수행자: OT Lead
- 검토자: HR
- 승인자: HR Director

## 3. 범위
- 연 1회 + 임시(전략 변경 시)

## 4. 입력 / 산출물
- Input: 직무기술서·전략·만족도
- Output: [[TMP-CMMI-05-01-02-01_교육요구_평가서_v1.0]]

## 5. 수행 절차

### 5.1 사전 준비
1. 직무 매핑.
2. 데이터 수집.

### 5.2 수행 단계
1. **요구 도출**.
2. **격차 분석**.
3. **우선순위**.
4. **검토**.
5. **승인**.
6. **계획 입력**.

### 5.3 완료 조건
- [ ] 평가서 승인본
- [ ] 격차 식별
- [ ] 계획 입력

## 6. 인터페이스 부서
- 부서장, GOV

## 7. 주의사항 / 예외 처리

### 7.1 자료 부족
- 인터뷰로 보완.

### 7.2 우선순위 충돌
- 전략 우선.

### 7.3 변경
- 전략 변경 시 갱신.
