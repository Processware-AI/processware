# 분기 거버넌스 검토 운영 (WI-CMMI-01-01-04)

> 상위 절차: [[PRO-CMMI-01-01_거버넌스_운영_절차_v1.0]]

## 1. 업무 목적
프로세스 구현·개선 활동·KPI·리스크·시정조치를 분기 단위로 경영진이 직접 검토하고 결정을 내려, 프로세스가 비즈니스 정렬을 유지하도록 한다.

## 2. 수행 주체
- **주 수행자**: SEPG Lead(간사)
- **참여자**: CEO(주재), 임원, PCB
- **승인자**: CEO

## 3. 범위
- 분기당 1회 정기 + 임시(중대 이벤트 시)
- 의제: KPI 현황·시정조치·리스크·자원 변경·신기술 도입

## 4. 입력 자료 / 산출물
- **Input**: KPI 보고, 시정조치 추적표, 리스크 등록부, [[PRO-CMMI-02-04_성과_및_측정_관리_절차_v1.0]] 산출 측정값
- **Output**: [[TMP-CMMI-01-01-04-01_거버넌스_회의록_v1.0]] 작성본, 결정사항 추적표

## 5. 수행 절차

### 5.1 사전 준비
1. 회의 14일 전 의제 수집 메일 발송.
2. 7일 전 자료 마감, SEPG 가 통합 패키지 작성.
3. 3일 전 사전 배포.

### 5.2 수행 단계
1. **개회 및 전 회의 결정사항 점검** — 미종결 항목 진척 확인.
2. **KPI 현황 보고** — PA별 임계값 위반 항목 식별.
3. **시정조치·리스크 보고** — 주요 항목 진척·리스크 신규/변경.
4. **의사결정** — 자원 재배정·우선순위 변경·시정조치 발의.
5. **회의록 작성** — 간사가 결정사항을 명확한 표현으로 기록.
6. **CEO 승인 후 배포** — 회의 종료 5영업일 내 승인·배포.
7. **이행 추적** — 결정사항을 추적표에 등재, 차회 회의에 진척 보고.

### 5.3 완료 조건
- [ ] 의사록 CEO 서명본 5영업일 내 배포
- [ ] 결정사항 추적표 등재
- [ ] KPI 임계값 위반 항목별 시정조치 발의
- [ ] 차회 회의 의제에 미종결 항목 자동 포함
- [ ] [[MAT-001_문서관리대장]] Row 갱신

## 6. 인터페이스 부서
- **PCB**: 의제 협의·심의
- **각 Process Owner**: KPI 보고
- **QA**: 부적합 보고

## 7. 주의사항 / 예외 처리

### 7.1 정족수 미달
- 임원 50% 미만 참석 시 회의 무효, 14일 내 재소집.

### 7.2 긴급 의제
- 중대 부적합·법규 변경 등은 임시 회의 7일 내 개최.

### 7.3 의사결정 보류
- 자료 부족 시 보류 가능, 차회 회의 1순위 의제로 자동 등재.
