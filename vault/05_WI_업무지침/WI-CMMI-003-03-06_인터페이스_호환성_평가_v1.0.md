# 인터페이스 호환성 평가 (WI-CMMI-003-03-06)

> 상위 절차: [[PRO-CMMI-303_제품통합_절차_v1.0]]

## 1. 업무 목적
통합 직전·중 인터페이스의 호환성을 정량 평가하여 호환 결함을 사전 차단한다.

## 2. 수행 주체
- 주 수행자: Integration Lead
- 검토자: 아키텍트
- 승인자: PM

## 3. 범위
- 모든 외부 인터페이스, 주요 내부 인터페이스

## 4. 입력 / 산출물
- Input: 인터페이스 명세
- Output: [[TMP-CMMI-003-03-06_인터페이스_호환성_보고서_v1.0]]

## 5. 수행 절차

### 5.1 사전 준비
1. 호환성 매트릭스.
2. 자동 테스트.

### 5.2 수행 단계
1. **계약 검증**.
2. **버전 매트릭스**.
3. **계약 테스트**.
4. **차이 분석**.
5. **시정 결정**.
6. **재평가**.

### 5.3 완료 조건
- [ ] 보고서
- [ ] 호환 매트릭스 통과
- [ ] 시정 종결

## 6. 인터페이스 부서
- 외부 벤더, 보안

## 7. 주의사항 / 예외 처리

### 7.1 Breaking
- 6개월 이중 지원.

### 7.2 외부 변경
- 30일 사전 통보.

### 7.3 보안
- 보안 검토 필수.
