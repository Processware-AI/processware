# 피어리뷰 준비·실행 (WI-CMMI-03-04-02)

> 상위 절차: [[PRO-CMMI-03-04_검증_절차]]

## 1. 업무 목적
주요 산출물(요구·설계·코드·문서)에 대해 피어리뷰를 준비하고 실시하여 결함을 조기에 식별·제거한다.

## 2. 수행 주체
- **주 수행자**: Peer Review Moderator (모더레이터)
- **검토자**: Peer Reviewer (저자 외 동료 2명 이상)
- **승인자**: Project Manager

## 3. 범위
PRO-CMMI-03-04 §5 의 **SP2.1 (Prepare for Peer Reviews) ~ SP2.2 (Conduct Peer Reviews)**.

## 4. 입력 자료 / 산출물
- **Input**
  - 검증 계획서 (WI-CMMI-03-04-01)
  - 리뷰 대상 산출물
  - 산출물별 체크리스트 (조직 표준)
- **Output**
  - [[TMP-CMMI-03-04-02-01_피어리뷰_체크리스트_기록부]] (체크리스트 + 결과)

## 5. 수행 절차

### 5.1 사전 준비
1. 산출물 유형별 체크리스트 확보 (예: 요구·설계·코드 각각).
2. 리뷰 형식 결정 — Inspection / Walkthrough / Pair / Tool-assisted.
3. 모더레이터·서기·리뷰어 배정.

### 5.2 수행 단계
1. **준비 (SP2.1)**
   - 리뷰 자료 배포 → 리뷰어 사전 학습 시간 부여.
   - 진입 기준 (산출물 완성도, 자체 점검) 점검.
   - 종료 기준(결함 등급별 한계 등) 사전 합의.
2. **리뷰 실행 (SP2.2)**
   - 모더레이터 진행, 서기 기록.
   - 결함 식별 (결함 vs 의견 구분).
   - 결함 등급 부여 (Major/Minor/Suggestion).
3. **결함 등록·할당**
   - 결함을 이슈 트래커에 등록 → 저자 시정.
4. **종료 판정**
   - 종료 기준 충족 → 리뷰 종료.
   - 미충족 → 재리뷰 일정.

### 5.3 완료 조건 체크리스트
- [ ] 산출물 유형별 체크리스트 사용
- [ ] 리뷰어 2명 이상 (저자 제외)
- [ ] 진입·종료 기준 사전 합의·기록
- [ ] 결함 vs 의견 구분 기록
- [ ] 결함 이슈 트래커 등록·할당
- [ ] 종료 판정 결재
- [ ] 리뷰 데이터(시간·결함 수 등) 측정 기록 — WI-03 분석 입력

## 6. 인터페이스 부서
- **VER 팀 + 산출물 저자 부서**: 주 수행
- **WI-CMMI-03-04-03 (분석)**: 리뷰 데이터 수신
- **CM**: 산출물 베이스라인 통제
- **PPQA**: 절차 준수 감사

## 7. 주의사항 / 예외 처리

### 7.1 진입 기준 미달
- 산출물 완성도 부족:
  - 리뷰 보류 + 저자 보완.
  - 무리한 진행 시 결함 산정 왜곡.

### 7.2 리뷰어 부족
- 적합한 리뷰어 부재(전문성·시간):
  - 외부 SME 초빙 (NDA 필수).
  - 비동기 회람 + 모더레이터 정리.

### 7.3 결함 vs 의견 분쟁
- 결함 분류 이견:
  - 모더레이터 1차 결정 → 미합의 시 PM 격상.
  - 결정 근거 회의록 기록.
