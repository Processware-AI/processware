# 직무교육 제공 (WI-CMMI-005-01-01)

> 상위 절차: [[PRO-CMMI-501_조직_훈련_절차_v1.0]]

## 1. 업무 목적
직무 수행에 필요한 교육을 신입·재직 인력에 적시 제공한다.

## 2. 수행 주체
- 주 수행자: OT Lead
- 검토자: HR
- 승인자: HR Director

## 3. 범위
- 신입 온보딩 + 재직 직무 갱신

## 4. 입력 / 산출물
- Input: 직무기술서, 격차 분석
- Output: [[TMP-CMMI-005-01-01_직무교육_시행계획서_v1.0]]

## 5. 수행 절차

### 5.1 사전 준비
1. 콘텐츠 검토.
2. 강사 섭외.

### 5.2 수행 단계
1. **계획**.
2. **수강 등록**.
3. **시행**.
4. **출석·평가**.
5. **수료**.
6. **이력 등재**.

### 5.3 완료 조건
- [ ] 시행 완료
- [ ] 수료자 ≥ 90%
- [ ] 이력 등재

## 6. 인터페이스 부서
- HR, 각 부서

## 7. 주의사항 / 예외 처리

### 7.1 미참석
- 재공지·보완 일정.

### 7.2 콘텐츠 노후
- 연 1회 갱신.

### 7.3 외부 강의
- 라이선스·NDA.
