# 측정 및 분석방법 정의 (WI-CMMI-02-04-03)

> 상위 절차: [[PRO-CMMI-02-04_성과_및_측정_관리_절차_v1.0]]

## 1. 업무 목적
측정 항목별 정의서(공식·단위·주기·도구·분석)를 표준화하여 측정의 일관성과 비교 가능성을 보장한다.

## 2. 수행 주체
- 주 수행자: MPM Lead
- 검토자: PCB, IT
- 승인자: PCB

## 3. 범위
- 신규 KPI, 갱신 KPI

## 4. 입력 / 산출물
- Input: 성과 목표, 데이터 가용성
- Output: [[TMP-CMMI-02-04-03-01_측정정의서_v1.0]]

## 5. 수행 절차

### 5.1 사전 준비
1. 데이터 소스 식별.
2. 분석 도구 선택.

### 5.2 수행 단계
1. **공식 정의**.
2. **단위·주기**.
3. **수집 절차**.
4. **분석 방법** — 통계·시각화.
5. **시범 운영**.
6. **확정**.

### 5.3 완료 조건
- [ ] 정의서 승인본
- [ ] 시범 운영 결과
- [ ] 데이터 사전 갱신

## 6. 인터페이스 부서
- IT, 각 부서

## 7. 주의사항 / 예외 처리

### 7.1 단위 혼용
- 절대 금지.

### 7.2 도구 변경
- 변경 시 정의서 갱신·재시범.

### 7.3 외부 비교
- 산업 비교 시 정의 일치 확인.
