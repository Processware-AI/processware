# 이해관계자·진행·마일스톤 검토 (WI-CMMI-02-02-02)

> 상위 절차: [[PRO-CMMI-02-02_프로젝트_모니터링_통제_절차]]

## 1. 업무 목적
이해관계자 참여 모니터링, 정기 진행 검토, 라이프사이클 단계 종료 시점의 마일스톤 검토를 수행하여 게이트 통과를 결정하고 다음 단계 진행을 승인한다.

## 2. 수행 주체
- **주 수행자**: Project Manager
- **검토자**: PMO Director, 이해관계자 전원
- **승인자**: Senior Management (마일스톤 통과)

## 3. 범위
PRO-CMMI-02-02 §5 의 **SP1.5(이해관계자) ~ SP1.7(마일스톤)**.

## 4. 입력 자료 / 산출물
- **Input**
  - 이해관계자 참여 계획서 ([[TMP-CMMI-02-01-03-02]])
  - WI-01 산출 주간 성과 기록·월간 약정 체크
  - 마일스톤 단계 산출물 (해당 시점)
- **Output**
  - 이해관계자 참여 기록 (별표, 매월)
  - [[TMP-CMMI-02-02-02-01_진행_검토_결과보고]] (월간)
  - [[TMP-CMMI-02-02-02-02_마일스톤_검토_결과보고]] (마일스톤별)

## 5. 수행 절차

### 5.1 사전 준비
1. 이해관계자 참여 계획서·참여 빈도 확인.
2. 진행 검토 회의 일정 (월 1회 권장).
3. 마일스톤 검토 회의 일정 (계획상 게이트 시점 ±5일).

### 5.2 수행 단계
1. **이해관계자 참여 모니터링 (SP1.5)** — 월 1회
   - 참여 계획서 vs 실제 참여 비교
   - 미참여 이해관계자 보강 협상
2. **진행 검토 (SP1.6)** — 월 1회
   - 베이스라인 대비 진척·예산·리스크·약정·이해관계자 status 종합
   - 결과를 Senior Mgmt 보고
3. **마일스톤 검토 (SP1.7)** — 마일스톤 시점
   - 마일스톤 통과 조건 (M1~M5 정의된 통과조건) 점검
   - 단계 산출물 audit (PPQA, CM 결재 포함)
   - 통과 결정: Pass / Conditional Pass / Fail
4. **게이트 결정·기록**
   - Pass 시 다음 단계 진행 승인
   - Conditional Pass: 조건부 진행 + 시정 마감 명시
   - Fail: 시정조치 후 재검토 (PMC SP2 분기)

### 5.3 완료 조건
- [ ] 이해관계자 참여 기록 매월 작성 — 미참여 0건 또는 보강 계획
- [ ] 월간 진행 검토 결과보고 — Senior Mgmt 결재
- [ ] 마일스톤 통과 조건 audit 통과 → 결정 (Pass/Conditional/Fail)
- [ ] 결정 사항 [[MAT-001]] 등록 + CM 베이스라인 (Conditional 이상)
- [ ] Fail 시 PMC SP2 시정조치 분기 진입

## 6. 인터페이스 부서
- **이해관계자 전원**: 회의 참석·결재
- **PPQA**: 마일스톤 audit ([[PRO-CMMI-04-03]])
- **CM**: 마일스톤 베이스라인 등록 ([[PRO-CMMI-04-01]])
- **Senior Mgmt**: 진행 검토·마일스톤 통과 승인

## 7. 주의사항 / 예외 처리

### 7.1 이해관계자 참여 저조
- 참여율 < 70% 시 PM 가 이해관계자별 1:1 outreach.
- 3개월 연속 저조 시 참여 계획 재수립 ([[WI-CMMI-02-01-03]] 재실행).

### 7.2 마일스톤 통과 조건 일부 미충족
- Conditional Pass 발급 — 시정 마감일 ≤ 2주 명시.
- 마감 미준수 시 Fail 로 재평가.

### 7.3 마일스톤 Fail
- 시정 후 재검토. 재검토 1회 추가 Fail 시 Sponsor escalation + 베이스라인 변경 검토.
