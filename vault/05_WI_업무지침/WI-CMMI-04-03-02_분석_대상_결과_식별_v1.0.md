# 분석 대상 결과 식별 (WI-CMMI-04-03-02)

> 상위 절차: [[PRO-CMMI-04-03_근본원인분석_및_해결_절차_v1.0]]

## 1. 업무 목적
근본원인분석(RCA) 이 필요한 결과(반복 결함·중대 사고 등) 를 식별·우선순위화한다.

## 2. 수행 주체
- 주 수행자: QA Lead
- 검토자: SEPG
- 승인자: PM

## 3. 범위
- 분기 1회 + 임시(중대 사고 시)

## 4. 입력 / 산출물
- Input: 결함·사고 데이터
- Output: [[TMP-CMMI-04-03-02-01_RCA_대상_식별표_v1.0]]

## 5. 수행 절차

### 5.1 사전 준비
1. 임계 기준.
2. 데이터.

### 5.2 수행 단계
1. **데이터 분석**.
2. **임계 위반 식별**.
3. **우선순위**.
4. **RCA 회부**.
5. **추적**.
6. **종결 검증**.

### 5.3 완료 조건
- [ ] 식별표
- [ ] 회부 완료
- [ ] 추적 시작

## 6. 인터페이스 부서
- 모든 팀

## 7. 주의사항 / 예외 처리

### 7.1 표본 부족
- 보조 정성 분석.

### 7.2 부서 회피
- PCB 보고.

### 7.3 외부 사고
- 외부 보고서 통합.
