# 프로세스 액션 플랜 수립·실행 (WI-CMMI-01-02-02)

> 상위 절차: [[PRO-CMMI-01-02_조직_프로세스_개선_배포_절차]]

## 1. 업무 목적
이전 단계에서 식별된 개선 권고를 실행 가능한 액션 플랜으로 구체화하여, 액션 추진팀(Process Action Team)을 구성하고 파일럿·실행 모니터링을 통해 합의된 결과물을 산출한다.

## 2. 수행 주체
- **주 수행자**: EPG Lead
- **검토자**: Process Action Team Lead, Process Owner
- **승인자**: CEO/CTO (액션 플랜 승인)

## 3. 범위
PRO-CMMI-01-02 §5 의 **SP2.1(액션 플랜 수립) ~ SP2.2(액션 실행)** 단계. 배포 단계는 [[WI-CMMI-01-02-03]] 참조.

## 4. 입력 자료 / 산출물
- **Input**
  - [[TMP-CMMI-01-02-01-03_개선_권고_목록]] 작성본 (우선순위 부여 완료)
  - OSSP, OPA 자원 가용성
  - 이해관계자 약정 의사
- **Output**
  - [[TMP-CMMI-01-02-02-01_프로세스_액션_플랜]] 작성본 (승인본)
  - [[TMP-CMMI-01-02-02-02_액션_추진_상태_기록부]] 작성본
  - 파일럿 결과보고서(필요 시)

## 5. 수행 절차

### 5.1 사전 준비
1. 개선 권고 목록의 승인 상태 확인 ([[WI-CMMI-01-02-01]] §5.3 결재 완료).
2. 액션 항목별 책임자 후보 사전 협의.
3. 가용 자원·예산 확인.

### 5.2 수행 단계
1. **액션 전략·접근법 정의** (SP2.1)
   - 각 개선 항목에 대해 실행 전략(전체 도입·단계 도입·파일럿) 선택.
   - 의존 관계·선후 정의.
2. **Process Action Team 구성** (SP2.1)
   - 항목별 PAT Lead·구성원·역할 지정.
   - 부서간 협조 약정서 체결.
3. **액션 플랜 문서화** (SP2.1)
   - [[TMP-CMMI-01-02-02-01]] 양식에 기록:
     - 액션 ID·목적·접근법·일정·자원·산출물·완료기준·리스크.
   - 이해관계자 리뷰 → 코멘트 반영.
4. **승인 및 약정 체결** (SP2.1)
   - CEO/CTO 승인.
   - PAT 구성원 약정 서명.
5. **파일럿 계획·실행** (SP2.2, 해당 시)
   - 신규·중대 개선은 파일럿 1회 이상 의무.
   - 파일럿 범위·기간·성공 기준 정의.
6. **액션 실행 및 추적** (SP2.2)
   - 주간 단위로 진행률·이슈 추적.
   - [[TMP-CMMI-01-02-02-02]] 상태 기록부 갱신.
   - 합동 리뷰(EPG + PAT + 이해관계자) 격주 또는 마일스톤별.
7. **이슈 관리 및 종결** (SP2.2)
   - 액션 수행 중 발생 이슈: 식별·문서화·종결까지 추적.
   - 완료 시 결과물(산출물) 검수 + 종결 확인서.

### 5.3 완료 조건
- [ ] 액션 플랜 작성·CEO/CTO 승인 완료
- [ ] PAT 구성·약정서 체결 완료
- [ ] 파일럿 계획·실행 완료 (해당 항목)
- [ ] 액션별 진행 상태 추적 기록 보존
- [ ] 합동 리뷰 회의록 저장
- [ ] 이슈 종결률 100% 또는 미종결 사유 명시
- [ ] CM 베이스라인 + MAT-001 등록

## 6. 인터페이스 부서
- **선행 WI**: [[WI-CMMI-01-02-01]] 개선 권고 입력
- **후행 WI**: [[WI-CMMI-01-02-03]] 배포 대상으로 액션 결과 인수인계
- **Process Owner**: 액션 결과 검수
- **PPQA**: 액션 활동 컴플라이언스 감사
- **CM**: 액션 산출물 베이스라인화

## 7. 주의사항 / 예외 처리

### 7.1 파일럿 실패
- 파일럿 결과가 성공 기준 미달 시: 액션 플랜 재설계 → 재승인.
- 2회 연속 실패 시 해당 개선 항목 폐기 + 사유 등재.

### 7.2 자원 부족·예산 미확보
- 액션 실행 중 자원 부족 발생 시: PAT Lead → EPG → Senior Mgmt 에스컬레이션.
- 우선순위 재조정 + 미실행 항목은 다음 사이클 이관.

### 7.3 이해관계자 약정 불이행
- 약정 부서가 협조 불이행 시: PCB 안건 상정 + 경영진 개입.
- 액션 일정 미달 영향 평가 + 재조정.
