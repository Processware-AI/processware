# 설계 기반 구현 (WI-CMMI-003-02-05)

> 상위 절차: [[PRO-CMMI-302_기술솔루션_절차_v1.0]]

승인된 설계에 따라 코드를 구현하고 설계 이탈을 즉시 발견·시정한다.

## 2. 수행 주체

- 검토자: Tech Lead
- 승인자: PM

- Output: 코드, [[TMP-CMMI-003-02-05_구현_체크리스트_v1.0]]

1. 설계 학습.
2. 코딩 표준 확인.

2. **단위검증 작성**.
3. **정적분석**.
