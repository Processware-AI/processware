# 결과 전달 및 운영관리 (WI-CMMI-002-04-06)

> 상위 절차: [[PRO-CMMI-204_성과_및_측정_관리_절차_v1.0]]

## 1. 업무 목적
분석 결과를 이해관계자에게 적시·적합 채널로 전달하고 KPI 대시보드를 운영한다.

## 2. 수행 주체
- 주 수행자: MPM Lead
- 검토자: PCB
- 승인자: MPM Lead

## 3. 범위
- KPI 대시보드 + 분기 보고서

## 4. 입력 / 산출물
- Input: 분석 리포트
- Output: [[TMP-CMMI-002-04-06_KPI_대시보드_사양서_v1.0]], 보고서

## 5. 수행 절차

### 5.1 사전 준비
1. 채널·대상 확정.
2. 권한 설정.

### 5.2 수행 단계
1. **대시보드 갱신**.
2. **분기 보고서 작성**.
3. **이해관계자 배포**.
4. **Q&A 운영**.
5. **개선 의견 수렴**.
6. **갱신**.

### 5.3 완료 조건
- [ ] 대시보드 가용성 ≥ 99%
- [ ] 분기 보고서 적시 배포
- [ ] Q&A 응답 SLA

## 6. 인터페이스 부서
- IT, 커뮤니케이션팀

## 7. 주의사항 / 예외 처리

### 7.1 대시보드 장애
- 24시간 내 복구.

### 7.2 데이터 노출
- 권한별 노출 범위 점검.

### 7.3 해석 오류
- 즉시 정정·재배포.
