# 위험·기회 분석 및 우선순위 (WI-CMMI-002-05-04)

> 상위 절차: [[PRO-CMMI-205_리스크_및_기회_관리_절차_v1.0]]

## 1. 업무 목적
정량 분석(확률·영향·노출)으로 위험·기회 우선순위를 결정한다.

## 2. 수행 주체
- 주 수행자: Risk Manager
- 검토자: PMO
- 승인자: PM

## 3. 범위
- 등록 후 14일 내 정량 분석

## 4. 입력 / 산출물
- Input: 등록부, 과거 데이터
- Output: [[TMP-CMMI-002-05-04_위험기회_분석표_v1.0]]

## 5. 수행 절차

### 5.1 사전 준비
1. 분석 척도 정의.
2. 데이터 확보.

### 5.2 수행 단계
1. **확률 평가**.
2. **영향 평가**.
3. **노출(P×I)**.
4. **우선순위 정렬**.
5. **임계값 비교**.
6. **회의 보고**.

### 5.3 완료 조건
- [ ] 분석표 완료
- [ ] 우선순위 정렬
- [ ] 등록부 갱신

## 6. 인터페이스 부서
- PMO, Sponsor

## 7. 주의사항 / 예외 처리

### 7.1 정성 척도 일관성
- 평가자별 척도 calibration 워크숍.

### 7.2 의존성
- 위험 간 의존성 별도 표시.

### 7.3 시간 변화
- 분기마다 재평가.
