# 예산 및 일정 개발 (WI-CMMI-002-01-04)

> 상위 절차: [[PRO-CMMI-201_프로젝트_계획_절차_v1.0]]

## 1. 업무 목적
WBS·추정 결과를 바탕으로 프로젝트 예산·일정을 개발·승인하여 실행 통제의 기준선을 확보한다.

## 2. 수행 주체
- 주 수행자: 프로젝트 PM
- 검토자: PMO, CFO
- 승인자: PM Sponsor(임원)

## 3. 범위
- 신규 프로젝트, 단계 게이트(Phase Gate) 갱신

## 4. 입력 / 산출물
- Input: WBS, [[PRO-CMMI-203_추정_관리_절차_v1.0]] 결과
- Output: [[TMP-CMMI-002-01-04_예산_일정_베이스라인_v1.0]] 승인본

## 5. 수행 절차

### 5.1 사전 준비
1. 추정 결과 검증.
2. 자원·일정 제약 식별.

### 5.2 수행 단계
1. **공수→일정 변환**.
2. **마일스톤·게이트 설정**.
3. **예산 산정** — 인건비·도구·외주 합산.
4. **버퍼·리스크 충당** — 10~20%.
5. **PMO·CFO 검토**.
6. **Sponsor 승인**.

### 5.3 완료 조건
- [ ] 베이스라인 승인본
- [ ] EVM 측정 시작점 등록
- [ ] [[MAT-001_문서관리대장]] Row

## 6. 인터페이스 부서
- PMO, 재무팀

## 7. 주의사항 / 예외 처리

### 7.1 예산 부족
- 부족 시 범위 축소·우선순위 조정·승인 후 재추진.

### 7.2 일정 압박
- 비현실적 일정 압박 시 PM 이 리스크 등록 후 Sponsor 보고.

### 7.3 환율·인플레
- 외화 항목은 환율 변동 시뮬레이션 첨부.
