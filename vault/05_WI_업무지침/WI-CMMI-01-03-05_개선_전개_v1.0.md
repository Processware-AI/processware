# 개선 전개 (WI-CMMI-01-03-05)

> 상위 절차: [[PRO-CMMI-01-03_프로세스_관리_및_개선_절차_v1.0]]

## 1. 업무 목적
파일럿에서 효과가 검증된 개선을 단계적으로 전사에 전개·정착시킨다.

## 2. 수행 주체
- 주 수행자: SEPG 전개팀
- 검토자: OT, QA, PCB
- 승인자: CEO

## 3. 범위
- 파일럿 KPI 충족 항목

## 4. 입력 / 산출물
- Input: 파일럿 결과, 교훈
- Output: [[TMP-CMMI-01-03-05-01_개선_전개계획서_v1.0]], 전개 보고서

## 5. 수행 절차

### 5.1 사전 준비
1. 영향 부서·시스템 식별.
2. 변화관리 자료 준비.

### 5.2 수행 단계
1. **전개 계획** — 단계·일정·교육·지표.
2. **사전 공지** — 영향 부서 30일 전.
3. **단계 1 적용** — 우선순위 부서.
4. **단계 2 적용** — 확산.
5. **전사 적용** — 90일 내 완료.
6. **정착 평가** — 90일 후.

### 5.3 완료 조건
- [ ] 전개 계획 승인본
- [ ] 전사 적용 완료
- [ ] 정착 평가 보고

## 6. 인터페이스 부서
- OT, QA, 각 부서

## 7. 주의사항 / 예외 처리

### 7.1 저항
- 저항 발생 시 변화관리 워크숍, 1:1 면담.

### 7.2 일정 지연
- 30% 이상 지연 시 PCB 보고.

### 7.3 정착 실패
- 90일 정착률 < 70% 시 재설계 검토.
