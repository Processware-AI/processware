# 인수 유형 결정·공급자 선정·협약 (WI-CMMI-02-04-01)

> 상위 절차: [[PRO-CMMI-02-04_공급자_협약_관리_절차]]

## 1. 업무 목적
외주/외부 인수 니즈에 대해 인수 유형 (COTS/OSS/맞춤개발/서비스/지급물) 을 결정하고, 공급자 후보를 평가·선정하여 협약 (SOW/Contract) 을 체결한다.

## 2. 수행 주체
- **주 수행자**: Procurement Lead
- **검토자**: PM, 도메인 엔지니어, 법무, 재무
- **승인자**: PMO Director (협약 결재)

## 3. 범위
PRO-CMMI-02-04 §5 의 **SG1 (SP1.1~1.3)**.

## 4. 입력 자료 / 산출물
- **Input**
  - 프로젝트 인수 니즈 (WBS 외주 항목)
  - 조직 공급자 카탈로그·이전 평가 기록
- **Output**
  - [[TMP-CMMI-02-04-01-01_인수_유형_결정서]]
  - [[TMP-CMMI-02-04-01-02_공급자_평가_선정_보고서]]
  - [[TMP-CMMI-02-04-01-03_공급자_협약서_SOW]]

## 5. 수행 절차

### 5.1 사전 준비
1. 인수 니즈 WBS 항목 추출.
2. 공급자 카탈로그·이전 평가 데이터 조회.

### 5.2 수행 단계
1. **인수 유형 결정 (SP1.1)** — COTS / OSS / 맞춤개발 / 서비스 / 고객 지급물 분류.
2. **공급자 평가 (SP1.2)** — 평가 기준 (기술·비용·일정·신뢰성·재무 건전성·법규) 으로 후보 비교.
3. **공급자 선정 (SP1.2)** — 최고 점수 + 법무·재무 사전 검토 통과 공급자 결정.
4. **협약 수립 (SP1.3)** — SOW (작업기술서), 일정·인도물·인수기준, 법적 조항 (보증·IP·기밀) 포함. 법무 결재.
5. **PMO Director 결재 + CM 등록**.

### 5.3 완료 조건
- [ ] 인수 유형 결정서 작성 — 분류 + 근거
- [ ] 공급자 평가 보고서 — 최소 3 후보 비교
- [ ] 공급자 협약서 (SOW + Contract) 체결
- [ ] 법무·재무 결재 첨부
- [ ] CM 등록 + MAT-001 등록

## 6. 인터페이스 부서
- **법무**: 협약 조항 검토 (A 권한)
- **재무**: 재무 건전성 평가·예산 승인
- **CM**: 협약서 베이스라인 등록

## 7. 주의사항 / 예외 처리

### 7.1 단일 공급자 (Sole Source)
- 시장 대안 없거나 기존 자산 보유 시 단일 공급자 허용 — 단, 사유 명시 + Sponsor 결재.

### 7.2 평가 점수 동률
- Tie-break 기준: 1) 재무 건전성, 2) 이전 평가 점수, 3) Sponsor 결정.

### 7.3 협약 협상 결렬
- 최종 후보 협상 결렬 시 차순위 공급자로 재협상.
- 차순위 부재 시 인수 유형 재검토 (예: 맞춤개발 → COTS 전환).
