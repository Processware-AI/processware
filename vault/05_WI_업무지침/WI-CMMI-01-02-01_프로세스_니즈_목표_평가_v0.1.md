# 프로세스 니즈·목표 및 평가 (WI-CMMI-01-02-01)

> 상위 절차: [[PRO-CMMI-01-02_조직_프로세스_개선_배포_절차]]

## 1. 업무 목적
조직의 비즈니스 목표·외부 표준·이해관계자 기대를 바탕으로 프로세스 니즈·목표를 정의하고, 정기·필요 시점에 프로세스를 평가하여 강점·약점·개선 기회를 식별한다. 본 결과는 액션 플랜의 입력이 된다.

## 2. 수행 주체
- **주 수행자**: EPG Lead
- **검토자**: Process Owner, Senior Management
- **승인자**: CEO/CTO (평가 결과 + 개선 권고 승인)

## 3. 범위
PRO-CMMI-01-02 §5 의 **SP1.1(니즈·목표) ~ SP1.3(개선 식별)** 단계에 적용. 액션 플랜 수립은 [[WI-CMMI-01-02-02]] 참조.

## 4. 입력 자료 / 산출물
- **Input**
  - 비즈니스 전략·연간 목표
  - 적용 표준(CMMI-DEV V1.3 ML3, ISO 9001 등)
  - 기존 OSSP·OPA, 전 주기 평가 결과
  - 이해관계자 기대·고객 클레임
- **Output**
  - [[TMP-CMMI-01-02-01-01_프로세스_니즈_목표_기술서]] 작성본
  - [[TMP-CMMI-01-02-01-02_프로세스_평가_계획_결과보고서]] 작성본
  - [[TMP-CMMI-01-02-01-03_개선_권고_목록]] 작성본 (우선순위 포함)

## 5. 수행 절차

### 5.1 사전 준비
1. 직전 평가(Appraisal) 결과·후속 액션 마감 상태 확인.
2. 비즈니스 전략 문서 최신본 확보 (Senior Mgmt 으로부터).
3. 평가 방법론 선정: SCAMPI A(공식) / B(중간) / C(빠른 진단) 중 평가 목적에 맞게 결정.
4. 평가 참여자·일정 사전 합의 (스폰서 확보).

### 5.2 수행 단계
1. **프로세스 니즈·목표 식별** (SP1.1)
   - 적용 정책·표준·비즈니스 목표 매핑 분석.
   - 프로세스 성능 목표(주기·결함밀도·고객만족 등)를 정량 지표로 정의.
   - 프로세스의 본질적 특성(예: 추적성·결정의 일관성) 도출.
   - [[TMP-CMMI-01-02-01-01]] 양식에 정리.
2. **평가 계획 수립** (SP1.2 준비)
   - Senior Mgmt 스폰서십 확보·발의문 확정.
   - 평가 범위(대상 PA·프로젝트·부서)·기간·예산 정의.
   - 평가팀 구성·교육(2일 이상 권장).
3. **평가 실행** (SP1.2)
   - 데이터 수집: 문서 검토·인터뷰·관찰.
   - 평가 기준 대비 격차(Gap) 식별.
   - 강점·약점·기회 분류.
4. **평가 결과 문서화** (SP1.2)
   - [[TMP-CMMI-01-02-01-02]] 양식에 결과 기록.
   - Senior Mgmt 보고 회의 개최.
5. **개선 기회 식별** (SP1.3)
   - 평가 결과로부터 후보 개선 항목 도출.
   - 우선순위 부여(영향도 × 실행가능성 매트릭스).
   - [[TMP-CMMI-01-02-01-03]] 양식에 정리.
   - 개선 권고 목록을 정기적으로 갱신(분기 단위).
6. **결과 보고 및 승인**
   - 평가 결과 + 개선 권고 목록을 패키지로 CEO/CTO 보고.
   - 승인 후 액션 플랜 단계로 이행 → [[WI-CMMI-01-02-02]].

### 5.3 완료 조건
- [ ] 프로세스 니즈·목표 기술서 작성·승인 완료
- [ ] 평가 계획(스폰서·범위·방법·일정) 결재 완료
- [ ] 평가 실시 + 결과보고서 작성 완료 (강점·약점·기회 명시)
- [ ] 개선 권고 목록 작성 + 우선순위 부여 완료 (최소 5건 이상)
- [ ] Senior Mgmt 보고 + CEO/CTO 승인 결재 첨부
- [ ] CM 베이스라인 등록 + MAT-001 등록

## 6. 인터페이스 부서
- **Senior Management**: 평가 스폰서십·전략 입력 (SP1.1 출처)
- **EPG**: 본 지침 주 수행
- **PPQA**: 평가 데이터 일부 공급(컴플라이언스 결과)
- **Process Owner**: 평가 협조 + 개선 권고 검토
- **다음 WI**: [[WI-CMMI-01-02-02]] 액션 플랜 수립

## 7. 주의사항 / 예외 처리

### 7.1 평가 주기 초과 (12개월 경과)
- 직전 평가 후 12개월 경과 시 즉시 SP1.2 발동.
- KPI "평가 주기" 미충족이 분기 보고서에 자동 플래그됨.

### 7.2 외부 강제 평가 (감독기관 등)
- 외부 감사·인증 평가가 도래하면 본 WI 의 자체 평가 일정을 조정하여 통합 운영.
- 외부 평가 결과는 SP1.2 결과보고서에 별첨으로 통합.

### 7.3 평가 결과 강한 부정 (Capability 미달)
- ML3 능력 미달 영역 발견 시: 액션 플랜 수립을 90일 이내로 단축.
- 사안 중대 시 별도 PCB 긴급 회의 소집.
