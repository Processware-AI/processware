# 초기 추정 수립 (WI-CMMI-02-03-01)

> 상위 절차: [[PRO-CMMI-02-03_추정_관리_절차_v1.0]]

## 1. 업무 목적
프로젝트 착수 단계의 정보로 1차 추정(규모·공수·일정·비용)을 수립하여 계획 입력으로 사용한다.

## 2. 수행 주체
- 주 수행자: PM
- 검토자: PMO, EST 전문가
- 승인자: PM

## 3. 범위
- 모든 신규 프로젝트, 단계 게이트

## 4. 입력 / 산출물
- Input: 헌장, 요구 개요, 과거 데이터
- Output: [[TMP-CMMI-02-03-01-01_초기_추정서_v1.0]]

## 5. 수행 절차

### 5.1 사전 준비
1. 유사 과거 프로젝트 검색.
2. 추정 모델 선택.

### 5.2 수행 단계
1. **규모 추정** — 기능점수·LoC.
2. **공수 환산** — 생산성 데이터 적용.
3. **일정 산출**.
4. **비용 산출**.
5. **불확실성 범위 부여** — ±30%.
6. **검토·승인**.

### 5.3 완료 조건
- [ ] 추정서 승인본
- [ ] 가정·매개변수 명시
- [ ] 불확실성 범위 부여

## 6. 인터페이스 부서
- PMO, EST

## 7. 주의사항 / 예외 처리

### 7.1 데이터 부족
- 과거 데이터 부족 시 전문가 판단(Wide-band Delphi).

### 7.2 정치적 압력
- 강제 조정 시 사유 기록.

### 7.3 갱신
- 단계 진행에 따라 점진적 추정(Progressive elaboration).
