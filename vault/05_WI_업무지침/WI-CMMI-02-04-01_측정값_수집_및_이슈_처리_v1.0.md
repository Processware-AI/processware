# 측정값 수집 및 이슈 처리 (WI-CMMI-02-04-01)

> 상위 절차: [[PRO-CMMI-02-04_성과_및_측정_관리_절차_v1.0]]

## 1. 업무 목적
정기 측정값을 표준 절차로 수집·검증·저장하고 이슈를 처리한다.

## 2. 수행 주체
- 주 수행자: MPM Lead
- 검토자: PMO
- 승인자: PM

## 3. 범위
- 주간/월간 측정 활동

## 4. 입력 / 산출물
- Input: 측정 정의, 도구 데이터
- Output: [[TMP-CMMI-02-04-01-01_측정값_수집_체크리스트_v1.0]] 완료본, 측정저장소

## 5. 수행 절차

### 5.1 사전 준비
1. 측정 정의 확인.
2. 자동화 스크립트 점검.

### 5.2 수행 단계
1. **수집** — 도구·수기.
2. **품질 점검** — 결측·이상치.
3. **변환·표준화**.
4. **이슈 처리** — 누락 보완·이상치 분리.
5. **저장**.
6. **알람** — 임계값 위반.

### 5.3 완료 조건
- [ ] 체크리스트 모든 ✅
- [ ] 저장소 적재
- [ ] 임계값 위반 시 알람

## 6. 인터페이스 부서
- IT, 각 팀

## 7. 주의사항 / 예외 처리

### 7.1 결측
- 결측 > 10% 시 사유 분석.

### 7.2 이상치
- 3시그마 초과 별도 분류.

### 7.3 도구 장애
- 수기 백업 절차.
