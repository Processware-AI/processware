# 작업산출물 평가 (WI-CMMI-004-01-03)

> 상위 절차: [[PRO-CMMI-401_프로세스_품질보증_절차_v1.0]]

## 1. 업무 목적
산출물 표본을 품질 기준 대비 평가하여 부적합을 식별한다.

## 2. 수행 주체
- 주 수행자: QA
- 검토자: Tech Lead
- 승인자: PM

## 3. 범위
- 단계별 표본

## 4. 입력 / 산출물
- Input: 산출물, 기준
- Output: [[TMP-CMMI-004-01-03_산출물_평가서_v1.0]]

## 5. 수행 절차

### 5.1 사전 준비
1. 표본 추출.
2. 루브릭.

### 5.2 수행 단계
1. **점검**.
2. **점수화**.
3. **부적합 등록**.
4. **시정 권고**.
5. **재점검**.
6. **종결**.

### 5.3 완료 조건
- [ ] 평가서
- [ ] 부적합 종결
- [ ] 데이터 적재

## 6. 인터페이스 부서
- 모든 팀

## 7. 주의사항 / 예외 처리

### 7.1 평가자 편향
- 2인 평가.

### 7.2 표본 부족
- 추가 표본.

### 7.3 자동화
- 도구 보완.
