# 솔루션 구축 기본 (WI-CMMI-003-02-01)

> 상위 절차: [[PRO-CMMI-302_기술솔루션_절차_v1.0]]

## 1. 업무 목적
설계→구현→단위검증의 기본 흐름을 표준화하여 솔루션 구축의 일관성을 확보한다.

## 2. 수행 주체
- 주 수행자: Dev Lead
- 검토자: 아키텍트
- 승인자: PM

## 3. 범위
- 모든 빌드 작업

## 4. 입력 / 산출물
- Input: 설계 문서, 요구
- Output: 구현물, 단위검증 결과

## 5. 수행 절차

### 5.1 사전 준비
1. 환경 셋업.
2. 코딩 표준 확인.

### 5.2 수행 단계
1. **설계 검토**.
2. **구현**.
3. **단위검증**.
4. **코드리뷰** — [[PRO-CMMI-305_동료검토_절차_v1.0]] 적용.
5. **머지**.
6. **CI 통과 확인**.

### 5.3 완료 조건
- [ ] 단위검증 통과
- [ ] 코드리뷰 승인
- [ ] CI 통과

## 6. 인터페이스 부서
- QA, CM

## 7. 주의사항 / 예외 처리

### 7.1 단위검증 미통과
- 머지 차단.

### 7.2 핫픽스
- QA·PM 사후 보고 의무.

### 7.3 외부 라이브러리
- 라이선스 점검.
