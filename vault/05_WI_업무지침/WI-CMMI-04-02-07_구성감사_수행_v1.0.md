# 구성감사 수행 (WI-CMMI-04-02-07)

> 상위 절차: [[PRO-CMMI-04-02_형상관리_절차_v1.0]]

## 1. 업무 목적
구성 항목의 정의·실제 일치성을 정기 감사하여 통제 무결성을 확인한다.

## 2. 수행 주체
- 주 수행자: CM Manager
- 검토자: QA
- 승인자: PM

## 3. 범위
- 분기 1회 + 릴리스 직전

## 4. 입력 / 산출물
- Input: CI 등록부, 실제 형상
- Output: [[TMP-CMMI-04-02-07-01_구성감사_보고서_v1.0]]

## 5. 수행 절차

### 5.1 사전 준비
1. 표본 추출.
2. 자동화 도구.

### 5.2 수행 단계
1. **기능 감사**.
2. **물리 감사**.
3. **일치성 검증**.
4. **부적합 등록**.
5. **시정**.
6. **재감사**.

### 5.3 완료 조건
- [ ] 보고서
- [ ] 부적합 종결
- [ ] 결과 적재

### 7.2 누락
