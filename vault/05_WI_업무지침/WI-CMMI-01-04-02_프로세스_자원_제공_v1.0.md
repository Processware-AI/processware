# 프로세스 자원 제공 (WI-CMMI-01-04-02)

> 상위 절차: [[PRO-CMMI-01-04_구현_인프라_운영_절차_v1.0]]

## 1. 업무 목적
프로세스 실행에 필요한 도구·템플릿·교육·코칭을 적시에 제공하여 실행 차질을 막는다.

## 2. 수행 주체
- 주 수행자: SEPG 운영자
- 검토자: IT, OT
- 승인자: SEPG Lead

## 3. 범위
- 진행 중 프로젝트 + 신규 부서

## 4. 입력 / 산출물
- Input: 자원 요청, 프로젝트 헌장
- Output: [[TMP-CMMI-01-04-02-01_프로세스_자원_제공_체크리스트_v1.0]] 완료본

## 5. 수행 절차

### 5.1 사전 준비
1. 프로젝트별 필요 자원 목록 확인.
2. 도구 라이선스 가용성 점검.

### 5.2 수행 단계
1. **요청 접수** — 자원 종류·수량·시점.
2. **승인** — SEPG Lead.
3. **배포** — 도구 라이선스·계정·문서 링크.
4. **온보딩** — 첫 1주 코칭.
5. **사용 모니터링** — 사용률 추적.
6. **회수** — 프로젝트 종료 시 라이선스 회수.

### 5.3 완료 조건
- [ ] 체크리스트 모든 항목 ✅
- [ ] 사용률 ≥ 80%
- [ ] 종료 시 회수 완료

## 6. 인터페이스 부서
- IT, 구매, OT

## 7. 주의사항 / 예외 처리

### 7.1 라이선스 부족
- 부족 시 우선순위 재배분 또는 추가 구매 발주.

### 7.2 사용률 저조
- 30일 사용률 < 50% 시 회수 또는 코칭 강화.

### 7.3 보안 사고
- 계정 공유·무단 사용 발견 시 즉시 회수, 보안팀 통보.
