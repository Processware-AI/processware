# 통합 절차 및 기준 (WI-CMMI-03-03-04)

> 상위 절차: [[PRO-CMMI-03-03_제품통합_절차_v1.0]]

## 1. 업무 목적
단계별 통합 진입·종료 기준을 명문화하여 통합의 품질 게이트를 확보한다.

## 2. 수행 주체
- 주 수행자: Integration Lead
- 검토자: QA
- 승인자: PM

## 3. 범위
- 통합 단계별

## 4. 입력 / 산출물
- Input: 통합 전략
- Output: [[TMP-CMMI-03-03-04-01_통합절차서_v1.0]]

## 5. 수행 절차

### 5.1 사전 준비
1. 단계 식별.
2. 게이트 후보.

### 5.2 수행 단계
1. **진입 기준 정의**.
2. **활동 절차**.
3. **종료 기준**.
4. **롤백 절차**.
5. **검토·승인**.
6. **공유**.

### 5.3 완료 조건
- [ ] 절차서 승인본
- [ ] 게이트 운영
- [ ] 롤백 검증

## 6. 인터페이스 부서
- QA, CM

## 7. 주의사항 / 예외 처리

### 7.1 게이트 우회
- 우회 발견 시 단계 무효화.

### 7.2 기준 충족 미흡
- 추가 시정 후 재진입.

### 7.3 외부 의존
- 외부 합의 시점 명시.
