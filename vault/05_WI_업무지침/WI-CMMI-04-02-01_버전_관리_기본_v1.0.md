- 검토자: CM Manager
- 승인자: PM

## 3. 범위
- 코드·문서·환경 설정

## 4. 입력 / 산출물

- Output: 커밋·태그·릴리스

1. 도구·계정.
2. 브랜치 정책.

1. **브랜치 생성**.
2. **개발·커밋**.
3. **PR 생성**.
4. **리뷰**.
5. **머지**.
6. **태그·릴리스**.
