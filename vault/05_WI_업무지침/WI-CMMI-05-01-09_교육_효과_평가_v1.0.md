# 교육 효과 평가 (WI-CMMI-05-01-09)

> 상위 절차: [[PRO-CMMI-05-01_조직_훈련_절차_v1.0]]

## 1. 업무 목적
Kirkpatrick 4단계(반응·학습·행동·결과) 로 교육 효과를 평가한다.

## 2. 수행 주체
- 주 수행자: OT Lead
- 검토자: HR
- 승인자: HR Director

## 3. 범위
- 연간 핵심 교육

## 4. 입력 / 산출물
- Input: 교육 데이터, KPI
- Output: [[TMP-CMMI-05-01-09-01_교육_효과평가서_v1.0]]

## 5. 수행 절차

### 5.1 사전 준비
1. 평가 설계.
2. 기준선 데이터.

### 5.2 수행 단계
1. **반응**(만족도).
2. **학습**(시험·과제).
3. **행동**(현업 적용).
4. **결과**(KPI 변화).
5. **분석**.
6. **권고**.

### 5.3 완료 조건
- [ ] 평가서
- [ ] 권고
- [ ] 차기 계획 입력

## 6. 인터페이스 부서
- 부서장, MPM

## 7. 주의사항 / 예외 처리

### 7.1 행동 평가 어려움
- 부서장 인터뷰.

### 7.2 외부 변수
- 분리 분석.

### 7.3 음(-) 효과
- 콘텐츠 재설계.
