# 개선 효과 평가 (WI-CMMI-001-03-06)

> 상위 절차: [[PRO-CMMI-103_프로세스_관리_및_개선_절차_v1.0)]]

## 1. 업무 목적
전개 완료 개선이 목표 KPI 를 달성했는지 정량 평가하여 종결 또는 보완·환류 결정을 내린다.

## 2. 수행 주체
- 주 수행자: SEPG 평가 담당
- 검토자: PCB
- 승인자: CEO

## 3. 범위
- 전개 후 90일·180일 시점 측정

## 4. 입력 / 산출물
- Input: 측정저장소 데이터, 사용자 만족도
- Output: [[TMP-CMMI-001-03-06_개선_효과평가서_v1.0]], 백로그 갱신

## 5. 수행 절차

### 5.1 사전 준비
1. 측정 시점·방법 사전 정의.
2. 기준선 데이터 준비.

### 5.2 수행 단계
1. **데이터 수집** — 측정저장소 + 만족도 조사.
2. **비교 분석** — 기준선 대비 변화량.
3. **유의성 검정** — 통계적 유의성 확인.
4. **결과 분류** — 성공/보완/실패.
5. **PCB 검토** — 성과·교훈 검토.
6. **CEO 결정** — 종결·보완·재기획.

### 5.3 완료 조건
- [ ] 효과평가서 승인본
- [ ] 백로그 갱신
- [ ] 교훈 PAL 등재

## 6. 인터페이스 부서
- MPM, OT, QA

## 7. 주의사항 / 예외 처리

### 7.1 데이터 부족
- 표본 < 30 시 보조 정성 분석 필수.

### 7.2 외부 변수
- 평가 기간 중 조직 개편 등 영향 분리 분석.

### 7.3 음(-) 효과
- KPI 악화 시 즉시 시정조치 발의.
