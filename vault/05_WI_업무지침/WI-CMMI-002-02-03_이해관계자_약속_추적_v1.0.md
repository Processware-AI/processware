# 이해관계자 약속 추적 (WI-CMMI-002-02-03)

> 상위 절차: [[PRO-CMMI-202_프로젝트_모니터링_및_통제_절차_v1.0]]

## 1. 업무 목적
프로젝트 내·외부 약속(인도일·범위·품질) 의 이행 상태를 추적·통제한다.

## 2. 수행 주체
- 주 수행자: PM
- 검토자: PMO
- 승인자: PM

## 3. 범위
- 모든 commitment(고객·내부)

## 4. 입력 / 산출물
- Input: 계약, 회의록, 메일
- Output: [[TMP-CMMI-002-02-03_약속_추적표_v1.0]]

## 5. 수행 절차

### 5.1 사전 준비
1. 약속 정의 및 등록 기준.
2. 추적표 초기화.

### 5.2 수행 단계
1. **등록** — 누가/무엇/언제까지.
2. **상태 갱신** — 주간.
3. **위반 알람**.
4. **재협상** — 위반 임박 시.
5. **종결** — 이행 확인 후.
6. **회고** — 종료 시.

### 5.3 완료 조건
- [ ] 추적표 갱신
- [ ] 위반 0건 또는 사후 조치 완료
- [ ] 회고 의사록

## 6. 인터페이스 부서
- 영업, 법무

## 7. 주의사항 / 예외 처리

### 7.1 위반
- 위반 시 즉시 이해관계자 통보, 재협상.

### 7.2 추가 약속
- 비공식 추가 약속도 등록 의무.

### 7.3 미이행
- 미이행 시 SLA 페널티 협의.
