# 측정저장소 저장 (WI-CMMI-02-04-05)

> 상위 절차: [[PRO-CMMI-02-04_성과_및_측정_관리_절차_v1.0]]

## 1. 업무 목적
프로젝트·조직 측정 데이터를 표준 스키마로 측정저장소에 적재한다.

## 2. 수행 주체
- 주 수행자: MPM Lead
- 검토자: SEPG, IT
- 승인자: MPM Lead

## 3. 범위
- 분기 적재 + 프로젝트 종료 시 1회 일괄 적재

## 4. 입력 / 산출물
- Input: 측정 결과
- Output: [[TMP-CMMI-02-04-05-01_저장소_적재_요청서_v1.0]] 완료본

## 5. 수행 절차

### 5.1 사전 준비
1. 스키마 확인.
2. 변환 스크립트.

### 5.2 수행 단계
1. **요청 접수**.
2. **품질 점검**.
3. **변환**.
4. **적재**.
5. **무결성 검증**.
6. **공표**.

### 5.3 완료 조건
- [ ] 요청서 보관
- [ ] 무결성 검증 통과
- [ ] 데이터 사전 갱신

## 6. 인터페이스 부서
- IT, SEPG

## 7. 주의사항 / 예외 처리

### 7.1 적재 실패
- 즉시 IT 통보, 재시도.

### 7.2 데이터 손상
- 백업에서 복구.

### 7.3 익명화
- 개인정보는 익명화 후 적재.
