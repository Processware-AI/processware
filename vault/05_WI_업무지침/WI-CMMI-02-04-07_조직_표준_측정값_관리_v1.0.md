# 조직 표준 측정값 관리 (WI-CMMI-02-04-07)

> 상위 절차: [[PRO-CMMI-02-04_성과_및_측정_관리_절차_v1.0]]

## 1. 업무 목적
모든 프로젝트가 공통으로 보고하는 표준 측정값(공통 KPI)을 정의·관리하여 비교 가능성과 조직 차원 분석을 가능케 한다.

## 2. 수행 주체
- 주 수행자: MPM Lead
- 검토자: PCB
- 승인자: CEO

## 3. 범위
- 연간 갱신 + 임시(전략 변경 시)

## 4. 입력 / 산출물
- Input: 비즈니스 전략, 프로젝트 KPI 후보
- Output: [[TMP-CMMI-02-04-07-01_표준_측정값_사전_v1.0]]

## 5. 수행 절차

### 5.1 사전 준비
1. 후보 KPI 수집.
2. 비교 가능성 평가.

### 5.2 수행 단계
1. **표준 KPI 선정**.
2. **정의서 작성**.
3. **시범 운영**.
4. **PCB 검토**.
5. **CEO 승인**.
6. **공표·교육**.

### 5.3 완료 조건
- [ ] 사전 승인본
- [ ] 시범 운영 결과
- [ ] 교육 완료

## 6. 인터페이스 부서
- 모든 프로젝트, OT

## 7. 주의사항 / 예외 처리

### 7.1 KPI 변경
- 변경 시 30일 사전 공지.

### 7.2 정의 충돌
- 충돌 시 CEO 결정.

### 7.3 신규 도메인
- 신규 도메인 진출 시 추가 KPI.
