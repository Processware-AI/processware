# 위험·기회 등록부 관리 (WI-CMMI-002-05-03)

> 상위 절차: [[PRO-CMMI-205_리스크_및_기회_관리_절차_v1.0]]

## 1. 업무 목적
위험·기회 항목의 식별~종결 전 생명주기를 등록부로 통합 관리한다.

## 2. 수행 주체
- 주 수행자: Risk Manager
- 검토자: PMO
- 승인자: PM

## 3. 범위
- 등록부 통합 관리

## 4. 입력 / 산출물
- Input: 식별·분석·대응 결과
- Output: [[TMP-CMMI-002-05-03_위험기회_등록부_v1.0]] 갱신

## 5. 수행 절차

### 5.1 사전 준비
1. 등록부 스키마 확인.
2. 권한 설정.

### 5.2 수행 단계
1. **항목 등록**.
2. **상태 갱신** — 주간.
3. **이력 보존**.
4. **알람 운영**.
5. **종결**.
6. **회고 입력**.

### 5.3 완료 조건
- [ ] 등록부 주간 갱신
- [ ] 이력 보존
- [ ] 종결 항목 회고 입력

## 6. 인터페이스 부서
- PMO, IT

## 7. 주의사항 / 예외 처리

### 7.1 데이터 손상
- 백업 복구.

### 7.2 권한 위반
- 즉시 회수.

### 7.3 종결 보류
- 사유 기록.
