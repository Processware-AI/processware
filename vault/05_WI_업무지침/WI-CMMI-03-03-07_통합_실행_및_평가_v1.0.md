# 통합 실행 및 평가 (WI-CMMI-03-03-07)

> 상위 절차: [[PRO-CMMI-03-03_제품통합_절차_v1.0]]

## 1. 업무 목적
통합을 실행하고 결과(결함·성능·인터페이스 무결성) 를 평가한다.

## 2. 수행 주체
- 주 수행자: Integration Lead
- 검토자: QA
- 승인자: PM

## 3. 범위
- 모든 통합 단계

## 4. 입력 / 산출물
- Input: 절차서·기준
- Output: [[TMP-CMMI-03-03-07-01_통합_실행_보고서_v1.0]]

## 5. 수행 절차

### 5.1 사전 준비
1. 게이트 통과 확인.
2. 모니터링 셋업.

### 5.2 수행 단계
1. **실행**.
2. **결함 식별**.
3. **성능 측정**.
4. **결과 평가**.
5. **시정**.
6. **베이스라인**.

### 5.3 완료 조건
- [ ] 결함 임계값 충족
- [ ] 보고서
- [ ] 베이스라인

## 6. 인터페이스 부서
- 개발, QA, CM

## 7. 주의사항 / 예외 처리

### 7.1 결함 다발
- 즉시 통합 중단·근본 원인 분석.

### 7.2 성능 저하
- 회귀 분석.

### 7.3 환경 차이
- 환경 표준화.
