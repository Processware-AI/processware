# 자원·지식·이해관계자 계획 + 통합 계획서 (WI-CMMI-02-01-03)

> 상위 절차: [[PRO-CMMI-02-01_프로젝트_계획_절차]]

## 1. 업무 목적
자원·지식·이해관계자 참여 계획을 수립하고, SP1.1~SP2.6 의 산출물을 단일 **통합 프로젝트 계획서** 로 묶어 SG3(약정) 단계에 인계한다.

## 2. 수행 주체
- **주 수행자**: Project Manager
- **검토자**: PMO Director, HR/OT Lead (지식·스킬), Stakeholder Manager
- **승인자**: CEO/CTO

## 3. 범위
PRO-CMMI-02-01 §5 의 **SP2.5(자원) ~ SP2.7(통합 계획서)**.

## 4. 입력 자료 / 산출물
- **Input**
  - [[WI-CMMI-02-01-01]]·[[WI-CMMI-02-01-02]] 모든 산출물
  - HR 가용 인력 풀, 조직도, OT 카탈로그 ([[PRO-CMMI-01-03]])
  - 이해관계자 매트릭스 (조직 표준)
- **Output**
  - [[TMP-CMMI-02-01-03-01_자원_지식_스킬_계획서]]
  - [[TMP-CMMI-02-01-03-02_이해관계자_참여_계획서]]
  - [[TMP-CMMI-02-01-03-03_통합_프로젝트_계획서]] (v0)

## 5. 수행 절차

### 5.1 사전 준비
1. WI-01·WI-02 산출물 전수 확보 (entry_gate 충족).
2. HR 시스템에서 가용 인력 query.
3. 이해관계자 스폰서·고객·내부 관계자 목록 초안 작성.

### 5.2 수행 단계
1. **자원 계획 (SP2.5)**
   - WBS × 일정 기반 자원 (인력·도구·시설) 배정
   - 자원 평준화(Resource Leveling) 분석
2. **지식·스킬 계획 (SP2.5)**
   - WP 별 필요 스킬셋 정의 (예: SwiftUI, AWS, ML/AI 등)
   - 현재 보유 스킬 vs 필요 스킬 갭 분석
   - 갭 해소 방법 명시: OT 신청 / 외부 교육 / 외주 / 채용
   - OT 신청 항목은 [[PRO-CMMI-01-03]] 입력 큐로 전달
3. **이해관계자 참여 계획 (SP2.6)**
   - 라이프사이클 단계별 참여 이해관계자 식별
   - 참여 빈도·채널·산출물 명시
   - RACI 매트릭스 작성 (별표)
4. **통합 프로젝트 계획서 작성 (SP2.7)**
   - WI-01~WI-03 모든 산출을 [[TMP-CMMI-02-01-03-03]] 양식으로 통합
   - cross-section consistency 체크 (예: 자원 vs 일정, 리스크 vs 예비비)
5. **베이스라인 후보화**
   - PMO + 인접 PA 책임자 review
   - WI-04 약정 획득으로 인계

### 5.3 완료 조건
- [ ] 자원·지식·스킬 계획서 작성 — 스킬 갭 해소 방안 명시
- [ ] OT 신청 항목 (있는 경우) [[PRO-CMMI-01-03]] 전달 완료
- [ ] 이해관계자 참여 계획서 — RACI + 참여 빈도·채널 명시
- [ ] 통합 프로젝트 계획서 v0 작성 — 단일 문서로 묶임
- [ ] cross-section consistency check 통과 (편차 0건)
- [ ] PMO + 인접 PA 책임자 review 통과

## 6. 인터페이스 부서
- **HR**: 자원 가용성 데이터 제공
- **OT (Organizational Training)**: 스킬 갭 입력 ([[PRO-CMMI-01-03]] SP1.1)
- **Stakeholder Manager**: 외부/고객 이해관계자 매트릭스 검증
- **MA**: 자원 활용률·스킬 갭 지표 등록

## 7. 주의사항 / 예외 처리

### 7.1 자원 부족 (allocation gap)
- 가용 인력 < 계획 효율 시: 1) 외주 검토 ([[PRO-CMMI-02-04]]), 2) 일정 연장 협의, 3) 범위 축소 협상.
- 모든 옵션 RSKM 등록 의무.

### 7.2 OT 일정 vs 프로젝트 일정 충돌
- 필요 스킬 OT 가 프로젝트 시작 후 제공되는 경우: 사전 교육·외부 교육으로 대체.
- 사전 학습 시간 1주 이상은 일정·예산에 반영.

### 7.3 이해관계자 누락 발견 (PMC 단계)
- PMC SP1.5 (이해관계자 참여 모니터링) 에서 신규 식별 시 본 WI 재실행.
- 빈번 누락 (분기 ≥ 2건) 시 식별 체크리스트 강화.
