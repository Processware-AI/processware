# 평가 방법 선정·대안 평가 (WI-CMMI-04-04-02)

> 상위 절차: [[PRO-CMMI-04-04_의사결정_분석_결정_절차]] · 표준: CMMI-DEV V1.3 DAR SG1 (중반)

## 1. 업무 목적
[[WI-CMMI-04-04-01]] 에서 정의한 평가 기준·대안에 대해 적절한 평가 방법(가중합산·AHP·실험·시뮬레이션 등)을 선정하고 대안을 평가하여 객관적 점수·근거를 산출한다.

## 2. 수행 주체
- **주 수행자**: Decision Facilitator
- **공동 수행**: 평가 패널
- **승인자**: Decision Owner

## 3. 범위
PRO-CMMI-04-04 §5 의 **SP1.4(평가 방법 선정) ~ SP1.5(대안 평가)** 적용. 해결안 선정은 [[WI-CMMI-04-04-03]] 참조.

## 4. 입력 자료 / 산출물
- **Input**
  - 평가 기준 정의서 ([[WI-CMMI-04-04-01]] 산출)
  - 대안 목록 ([[WI-CMMI-04-04-01]] 산출)
  - 사안 관련 정보·과거 사례
- **Output**
  - [[TMP-CMMI-04-04-02-01_평가방법_선정서]]
  - [[TMP-CMMI-04-04-02-02_대안_평가_결과서]]

## 5. 수행 절차

### 5.1 사전 준비
1. 평가 패널 확정(독립성 검토).
2. 평가 도구(스프레드시트·AHP 도구·실험 환경) 준비.
3. 사례·벤치마크 자료 수집.

### 5.2 수행 단계
1. **평가 방법 후보 도출 (SP1.4 전반)**
   - 가중합산(Weighted Sum), AHP(Analytic Hierarchy Process), 실험·PoC, 시뮬레이션, 시나리오 분석 등.
   - 사안 특성·자원·일정에 따른 적합도 평가.
2. **평가 방법 선정 (SP1.4 후반)**
   - 후보 중 1개 또는 조합 선정.
   - 선정 근거(객관성·재현성·비용) 명기.
3. **평가 수행 (SP1.5 전반)**
   - 대안별 기준별 점수 산출.
   - 가중치 적용 시 정량 점수 계산.
   - 정성 평가는 패널 합의 점수 사용.
4. **민감도 분석**
   - 가중치를 ±10% 변동 시 순위 변동 확인.
   - 순위 안정성 확인.
5. **평가 결과 정리 (SP1.5 후반)**
   - 대안별 종합 점수·순위·강약점 정리.
   - 각 점수의 근거·증적 보존.
6. **검토·승인**
   - Decision Owner 검토 → 결재.

### 5.3 완료 조건
- [ ] 평가 방법 선정 근거 명기
- [ ] 모든 대안이 모든 기준에 대해 평가
- [ ] 가중치 적용 시 계산 검증
- [ ] 민감도 분석 수행
- [ ] 점수 근거·증적 보존
- [ ] Decision Owner 결재

- **TS·OPF 등**: 기술적 자문 (해당 시)

## 7. 주의사항 / 예외 처리

### 7.1 평가 점수 편향 의심
- 평가자 1인의 점수가 다른 평가자 대비 ±2σ 이상 일탈:
  - 해당 평가자 점수 사유 확인 → 합의 시도.
  - 미합의 시 평균에서 일탈치 제외 또는 패널 재구성.

### 7.2 평가 도구 한계
- AHP 계산에서 일관성 비율(CR) > 0.1 인 경우:
  - 비교 행렬 재수정.
  - 재수정해도 일관성 미달 시 가중합산으로 전환.

### 7.3 새로운 정보 등장
- 평가 진행 중 새로운 정보 등장으로 기준·가중치 변경 필요 시:
  - [[WI-CMMI-04-04-01]] 으로 환류 → 기준 재정의 후 재평가.
