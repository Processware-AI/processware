# 분류·매개변수 관리 (WI-CMMI-02-05-07)

> 상위 절차: [[PRO-CMMI-02-05_리스크_및_기회_관리_절차_v1.0]]

## 1. 업무 목적
위험 분류 체계와 평가 매개변수(확률·영향 척도) 를 표준화·갱신한다.

## 2. 수행 주체
- 주 수행자: Risk Manager
- 검토자: PCB
- 승인자: CEO

## 3. 범위
- 연 1회 정기, 임시 시 갱신

## 4. 입력 / 산출물
- Input: 산업 모범사례, 사고 통계
- Output: [[TMP-CMMI-02-05-07-01_위험_분류매개변수표_v1.0]]

## 5. 수행 절차

### 5.1 사전 준비
1. 현행 분류·척도 검토.
2. 외부 벤치마크.

### 5.2 수행 단계
1. **분류 체계 갱신**.
2. **척도 정의 갱신**.
3. **시범 적용**.
4. **PCB 검토**.
5. **CEO 승인**.
6. **공표·교육**.

### 5.3 완료 조건
- [ ] 표 승인본
- [ ] 시범 결과
- [ ] 교육 완료
