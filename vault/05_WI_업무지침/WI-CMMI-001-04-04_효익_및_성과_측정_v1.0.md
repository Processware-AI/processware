# 효익 및 성과 측정 (WI-CMMI-001-04-04)

> 상위 절차: [[PRO-CMMI-104_구현_인프라_운영_절차_v1.0]]

## 1. 업무 목적
프로세스 운영이 비즈니스 효익(생산성·품질·일정·비용) 에 기여한 정도를 정량 측정·보고하여, 자원 투자 정당성을 입증한다.

## 2. 수행 주체
- 주 수행자: SEPG 측정 담당
- 검토자: MPM, PCB
- 승인자: CEO

## 3. 범위
- 분기 효익 보고 + 연간 종합 보고

## 4. 입력 / 산출물
- Input: 측정저장소, 프로젝트 결산
- Output: [[TMP-CMMI-001-04-04_프로세스_효익_보고서_v1.0]] 승인본

## 5. 수행 절차

### 5.1 사전 준비
1. 효익 지표 정의(ROI, NPV, 결함밀도 등).
2. 기준선 데이터 확보.

### 5.2 수행 단계
1. **데이터 수집** — 분기 측정값.
2. **지표 산출** — 표준 공식 적용.
3. **벤치마크 비교** — 산업 평균 대비.
4. **요인 분석** — 효익 기여·저해 요인.
5. **보고서 작성** — PCB·CEO 대상.
6. **공표** — 사내 공지.

### 5.3 완료 조건
- [ ] 보고서 승인본
- [ ] 사내 공지
- [ ] 차기 KPI 입력

## 6. 인터페이스 부서
- 재무팀, MPM, 각 부서

## 7. 주의사항 / 예외 처리

### 7.1 데이터 결측
- 분기 결측률 > 10% 시 보조 정성 분석 추가.

### 7.2 외부 변수
- 시장·환경 영향 분리하여 보고.

### 7.3 음(-) 효익
- KPI 악화 시 즉시 시정조치 발의.
