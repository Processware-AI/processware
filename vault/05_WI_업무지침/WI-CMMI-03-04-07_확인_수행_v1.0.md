# 확인 수행 (WI-CMMI-03-04-07)

> 상위 절차: [[PRO-CMMI-03-04_검증_및_확인_절차_v1.0]]

## 1. 업무 목적
운영 환경 또는 유사 환경에서 사용자 의도 충족을 확인한다.

## 2. 수행 주체
- 주 수행자: QA, 사용자
- 검토자: PM
- 승인자: Sponsor

## 3. 범위
- 사용자 인수 단계

## 4. 입력 / 산출물
- Input: 시나리오, 운영 환경
- Output: [[TMP-CMMI-03-04-07-01_확인_보고서_v1.0]]

## 5. 수행 절차

### 5.1 사전 준비
1. 환경.
2. 사용자 일정.

### 5.2 수행 단계
1. **사용자 테스트**.
2. **수용성 평가**.
3. **결함 식별**.
4. **시정**.
5. **재확인**.
6. **인수**.

### 5.3 완료 조건
- [ ] 인수 서명
- [ ] 결함 종결
- [ ] 보고서 등재

## 6. 인터페이스 부서

### 7.2 환경 차이
