# 기본 확인 수행 (WI-CMMI-003-04-02)

> 상위 절차: [[PRO-CMMI-304_검증_및_확인_절차_v1.0]]

## 1. 업무 목적
솔루션이 사용자 의도·운영 환경에서 원하는 결과를 내는지 확인한다.

## 2. 수행 주체
- 주 수행자: QA, 사용자
- 검토자: PM
- 승인자: PM

## 3. 범위
- 운영 환경 또는 유사 환경

## 4. 입력 / 산출물
- Input: 사용자 시나리오
- Output: [[TMP-CMMI-003-04-02_확인_체크리스트_v1.0]]

## 5. 수행 절차

### 5.1 사전 준비
1. 사용자 환경.
2. 시나리오.

### 5.2 수행 단계
1. **사용자 테스트**.
2. **수용성 평가**.
3. **결함 식별**.
4. **시정**.
5. **재확인**.
6. **인수**.

### 5.3 완료 조건
- [ ] 인수 서명
- [ ] 결함 종결
- [ ] 데이터 적재

## 6. 인터페이스 부서
- 운영, 사용자

## 7. 주의사항 / 예외 처리

### 7.1 사용자 가용성
- 사용자 일정 우선.

### 7.2 운영 환경 차이
- 차이 분리 분석.

### 7.3 부분 인수
- 범위 명시.
