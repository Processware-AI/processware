# 공급자 협약 실행·인수 (WI-CMMI-02-04-02)

> 상위 절차: [[PRO-CMMI-02-04_공급자_협약_관리_절차]]

## 1. 업무 목적
협약 체결된 공급자의 진행 상황을 모니터링하고, 인수 시점에 인수 기준 (SOW) 대비 제품을 검토·시험하여 수락/반려를 결정한다.

## 2. 수행 주체
- **주 수행자**: Procurement Lead (실행 모니터링), Project Manager (인수)
- **검토자**: 도메인 엔지니어, QA
- **승인자**: Project Manager (인수 결정)

## 3. 범위
PRO-CMMI-02-04 §5 의 **SG2 SP2.1~2.2**.

## 4. 입력 자료 / 산출물
- **Input**: 공급자 협약서, 공급자 마일스톤 일정, 인수 기준 (SOW §2.4)
- **Output**: [[TMP-CMMI-02-04-02-01]], [[TMP-CMMI-02-04-02-02]]

## 5. 수행 절차

### 5.1 사전 준비
1. 공급자 협약서 인수 기준 확인.
2. 인수 시험 환경 준비.

### 5.2 수행 단계
1. **협약 실행 모니터링 (SP2.1)** — 공급자 마일스톤별 진행 상황 추적, 정기 status 회의 (주/격주).
2. **인수 검토 (SP2.2)** — 공급자 인도물 받아 SOW 인수 기준 대비 평가.
3. **인수 시험 (SP2.2)** — 인도물에 대해 기능·성능·보안 시험. QA 참여.
4. **수락/반려 결정** — Pass 시 인수 완료; 부적합 시 공급자에 시정 요청.

### 5.3 완료 조건
- [ ] 공급자 진행 보고서 정기 작성 (주/격주)
- [ ] 인수 검토 및 시험 결과 기재
- [ ] 인수 기준 100% 충족 또는 부적합 시정 완료
- [ ] PM 수락 결재
- [ ] CM 등록 (인수 완료 산출물)

## 6. 인터페이스 부서
- **QA**: 인수 시험 참여 ([[PRO-CMMI-04-03]])
- **CM**: 인수 산출물 베이스라인 등록 ([[PRO-CMMI-04-01]])
- **법무**: 인수 시 SOW 분쟁 발생 시
- **PMC**: 모니터링 결과 PMC 반영 ([[WI-CMMI-02-02-01]])

## 7. 주의사항 / 예외 처리

### 7.1 공급자 지연
- 마일스톤 지연 발생 시 협약 §벌칙 조항 발동.
- 누적 지연 ≥ 2주 시 PMO escalation + 인수 일정 재조정 (PP 영향평가).

### 7.2 인수 부적합 (Fail)
- 1차 Fail: 공급자 시정 요청 (마감 명시).
- 2차 Fail: 협약 §벌칙 + 계약 종료 검토 (법무 협의).

### 7.3 SOW 변경 필요
- 인수 과정에서 SOW 변경 필요 발견 시: 공식 변경 협상 후 SOW 개정 → 법무·재무 재결재.
