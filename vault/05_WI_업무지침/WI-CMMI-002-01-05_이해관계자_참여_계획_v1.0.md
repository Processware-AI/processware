# 이해관계자 참여 계획 (WI-CMMI-002-01-05)

> 상위 절차: [[PRO-CMMI-201_프로젝트_계획_절차_v1.0]]

## 1. 업무 목적
프로젝트 결과에 영향을 주거나 받는 이해관계자를 식별하고 적정 관여 수준·소통 계획을 수립하여 의사소통 부재로 인한 갈등을 예방한다.

## 2. 수행 주체
- 주 수행자: 프로젝트 PM
- 검토자: PMO
- 승인자: 프로젝트 PM

## 3. 범위
- 모든 신규 프로젝트, 단계 게이트 갱신

## 4. 입력 / 산출물
- Input: 프로젝트 헌장, 조직도
- Output: [[TMP-CMMI-002-01-05_이해관계자_참여계획서_v1.0]] 승인본

## 5. 수행 절차

### 5.1 사전 준비
1. 이해관계자 후보 브레인스토밍.
2. 권력·관심 매트릭스 준비.

### 5.2 수행 단계
1. **식별** — 내부·외부 모두.
2. **분류** — 권력·관심 4사분면.
3. **관여 전략** — Manage closely / Keep satisfied 등.
4. **소통 계획** — 채널·빈도·내용.
5. **기록** — 참여계획서.
6. **분기 갱신**.

### 5.3 완료 조건
- [ ] 참여계획서 승인본
- [ ] 소통 채널 셋업
- [ ] 분기 갱신 일정

## 6. 인터페이스 부서
- 영업·CS·법무

## 7. 주의사항 / 예외 처리

### 7.1 신규 이해관계자
- 진행 중 신규 식별 시 즉시 등록·계획 갱신.

### 7.2 충돌
- 이해관계자 간 충돌 시 PM 이 중재·기록.

### 7.3 비공개 정보
- 일부 이해관계자에 대한 정보는 비공개 부속서로 분리.
