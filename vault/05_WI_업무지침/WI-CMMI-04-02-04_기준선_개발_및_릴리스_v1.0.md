# 기준선 개발 및 릴리스 (WI-CMMI-04-02-04)

> 상위 절차: [[PRO-CMMI-04-02_형상관리_절차_v1.0]]

승인된 산출물 집합을 기준선으로 고정·릴리스하여 후속 변경 통제 기반을 마련한다.

- 검토자: PM, QA
- 승인자: PM

1. 진입 기준 충족.
2. 태그 규칙.

2. **태그·릴리스 노트**.
