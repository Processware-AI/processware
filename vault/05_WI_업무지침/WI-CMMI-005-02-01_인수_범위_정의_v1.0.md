# 인수 범위 정의 (WI-CMMI-005-02-01)

> 상위 절차: [[PRO-CMMI-502_공급자_합의_관리_절차_v1.0]]

## 1. 업무 목적
공급자 인수 대상의 범위·요구사항·인도물을 명확히 정의한다.

## 2. 수행 주체
- 주 수행자: 구매팀
- 검토자: PM, 법무
- 승인자: PM

## 3. 범위
- 외부 인수 모든 항목

## 4. 입력 / 산출물
- Input: 비즈니스 요구
- Output: [[TMP-CMMI-005-02-01_인수_범위서_v1.0]]

## 5. 수행 절차

### 5.1 사전 준비
1. 비즈니스 요구.
2. 내부 vs 외부.

### 5.2 수행 단계
1. **범위 도출**.
2. **요구 명세**.
3. **인도물**.
4. **검토**.
5. **승인**.
6. **RFP 입력**.

### 5.3 완료 조건
- [ ] 범위서 승인본

- [ ] RFP 입력
