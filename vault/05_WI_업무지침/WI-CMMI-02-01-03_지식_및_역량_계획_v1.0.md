# 지식 및 역량 계획 (WI-CMMI-02-01-03)

> 상위 절차: [[PRO-CMMI-02-01_프로젝트_계획_절차_v1.0]]

## 1. 업무 목적
프로젝트 작업 수행에 필요한 지식·역량 격차를 사전 식별하고 교육·코칭·외부 인력 계획을 수립한다.

## 2. 수행 주체
- 주 수행자: 프로젝트 PM
- 검토자: HR, 부서장
- 승인자: 프로젝트 PM

## 3. 범위
- 모든 신규 프로젝트, 신규 인력 합류 시 갱신

## 4. 입력 / 산출물
- Input: WBS, 인력 풀, 직무기술서
- Output: [[TMP-CMMI-02-01-03-01_프로젝트_역량계획표_v1.0]] 승인본

## 5. 수행 절차

### 5.1 사전 준비
1. 작업별 요구 역량 식별.
2. 인력별 역량 자료 수집.

### 5.2 수행 단계
1. **역량 매트릭스 작성**.
2. **격차 식별**.
3. **해소 옵션 결정**(교육·코칭·채용·외주).
4. **HR 협의**.
5. **계획 등록**.
6. **분기 갱신**.

### 5.3 완료 조건
- [ ] 역량계획표 승인본
- [ ] 교육 신청 등록
- [ ] 분기 갱신 일정 설정

## 6. 인터페이스 부서
- HR, OT

## 7. 주의사항 / 예외 처리

### 7.1 역량 부족 인력
- 부족 인력에 대해 작업 단순화 또는 멘토링 필수.

### 7.2 외부 인력
- 외부 컨설턴트 활용 시 [[PRO-CMMI-05-02_공급자_합의_관리_절차_v1.0]] 적용.

### 7.3 갑작스런 결원
- 결원 시 24시간 내 backup 가동.
