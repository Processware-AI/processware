# 측정 및 성과 목표 도출 (WI-CMMI-002-04-02)

> 상위 절차: [[PRO-CMMI-204_성과_및_측정_관리_절차_v1.0]]

## 1. 업무 목적
비즈니스 목표를 측정 가능한 성과 목표로 분해하여 KPI 체계의 출발점을 정의한다.

## 2. 수행 주체
- 주 수행자: MPM Lead
- 검토자: PCB
- 승인자: Sponsor

## 3. 범위
- 연간 + 분기 갱신

## 4. 입력 / 산출물
- Input: 비즈니스 전략, 프로젝트 헌장
- Output: [[TMP-CMMI-002-04-02_측정_성과목표_도출표_v1.0]]

## 5. 수행 절차

### 5.1 사전 준비
1. 전략 목표 분석.
2. 측정 가능성 체크.

### 5.2 수행 단계
1. **목표 분해** — Strategy Map.
2. **KPI 매핑** — SMART.
3. **임계값 설정**.
4. **수단 결정** — 측정 도구·주기.
5. **검토 승인**.
6. **공표**.

### 5.3 완료 조건
- [ ] 도출표 승인본
- [ ] KPI 측정 정의 완료
- [ ] 공표

## 6. 인터페이스 부서
- 전략기획팀, MPM

## 7. 주의사항 / 예외 처리

### 7.1 측정 불가 목표
- 측정 불가 시 대용지표(Proxy) 정의.

### 7.2 KPI 충돌
- 충돌 KPI 는 가중치로 균형.

### 7.3 갱신
- 전략 변경 시 30일 내 갱신.
