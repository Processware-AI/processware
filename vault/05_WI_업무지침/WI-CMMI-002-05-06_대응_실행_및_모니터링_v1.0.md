# 대응 실행 및 모니터링 (WI-CMMI-002-05-06)

> 상위 절차: [[PRO-CMMI-205_리스크_및_기회_관리_절차_v1.0]]

## 1. 업무 목적
대응 계획을 실행하고 트리거 모니터링·종결을 운영한다.

## 2. 수행 주체
- 주 수행자: Risk Manager
- 검토자: PMO
- 승인자: PM

## 3. 범위
- High/Medium 항목

## 4. 입력 / 산출물
- Input: 대응 계획서
- Output: [[TMP-CMMI-002-05-06_대응_실행_보고서_v1.0]]

## 5. 수행 절차

### 5.1 사전 준비
1. 트리거 모니터링 가동.
2. 자원 준비.

### 5.2 수행 단계
1. **실행** — 트리거 발생 시.
2. **자원 동원**.
3. **모니터링**.
4. **효과 평가**.
5. **종결 또는 재계획**.
6. **회고**.

### 5.3 완료 조건
- [ ] 실행 보고서
- [ ] 종결 또는 재계획
- [ ] 회고 등재

## 6. 인터페이스 부서
- 모든 책임 부서

## 7. 주의사항 / 예외 처리

### 7.1 트리거 미작동
- 점검·교정.

### 7.2 효과 미흡
- 재계획.

### 7.3 자원 변경
- 변경 시 계획 수정.
