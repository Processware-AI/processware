# 추정치 대비 실적 추적 (WI-CMMI-002-02-02)

> 상위 절차: [[PRO-CMMI-202_프로젝트_모니터링_및_통제_절차_v1.0]]

## 1. 업무 목적
추정 vs 실제 격차를 정기 추적하여 추정 모델 개선과 통제 의사결정에 활용한다.

## 2. 수행 주체
- 주 수행자: PM
- 검토자: PMO
- 승인자: PM

## 3. 범위
- 격주 1회

## 4. 입력 / 산출물
- Input: 추정 결과, 실제 실적
- Output: [[TMP-CMMI-002-02-02_추정실적_추적표_v1.0]]

## 5. 수행 절차

### 5.1 사전 준비
1. 추정 baseline 확인.
2. 실적 데이터 추출.

### 5.2 수행 단계
1. **공수 차이 산출**.
2. **일정 SPI 산출**.
3. **비용 CPI 산출**.
4. **규모 차이**.
5. **원인 분석**.
6. **시정 조치 또는 추정 재조정**.

### 5.3 완료 조건
- [ ] 추적표 갱신
- [ ] CPI/SPI 임계값 위반 시 시정
- [ ] 측정저장소 적재

## 6. 인터페이스 부서
- PMO, EST

## 7. 주의사항 / 예외 처리

### 7.1 차이 ≥ 20%
- 즉시 PCB 보고.

### 7.2 외부 변수
- 외부 변수 영향은 분리 분석.

### 7.3 데이터 일관성
- 단위·시점 일관성 검증.
