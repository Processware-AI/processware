# 00_MOC — Map of Content

## 역할
Obsidian 탐색의 **허브/랜딩 페이지**. 개별 노트들을 주제별로 묶어 빠른 진입점 제공.

## 현재 MOC
| 파일 | 역할 |
|---|---|
| [[MOC_전체표준]] | 편입된 표준의 전체 상태판 (✅/🟡/⛔) |
| [[MOC_프로세스맵]] | 3-Tier(M/C/S) 프로세스 아키텍처 허브 |
| [[MOC_추적성매트릭스]] | MAT-001~009 (9종 운영, MAT-010 예약) + 표준별 추적성(MAT-011~) 진입점 |

## 무엇이 들어가나
- 주제별 허브 노트만
- 파일명 `MOC_{주제}.md` 일관 유지
- 상위에 링크만, 본문은 별도 문서에

## 무엇이 들어가지 않나
- 실제 산출물(POL/PRO/WI 등) → 해당 유형 폴더로
- 작업 기록·초안 → 02_표준 또는 해당 유형 폴더

## 관리 규칙
- 새 영역 확장 시 MOC 추가 (`MOC_{신영역}.md`)
- Dataview 플러그인으로 자동 집계 가능
- 편집은 PCB 심의 대상 아님 (링크 집합이므로)
