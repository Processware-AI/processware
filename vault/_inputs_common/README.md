# 전사 공통 입력자료

모든 표준 분석 시 **배경 컨텍스트**로 참조되는 조직 공통 자산. 표준별 `_inputs/` 와 같은 5카테고리 구조.

## 구조
```
_inputs_common/
├── 01_표준원문/        ← 전사에 공통 적용되는 기본 표준
├── 02_법규/            ← 사업 전반 관련 법령 (업종 공통)
├── 03_해설서/          ← 조직 차원 표준 해설 매뉴얼
├── 04_AsIs/            ← 전사 기존 자산 (조직도·미션·공통 POL)
└── 05_산업가이드/     ← 업종 공통 프레임워크
```

## 배치 권장
| 카테고리 | 들어갈 만한 것 |
|---|---|
| 01_표준원문 | 전사 기본 ISO 9001, 27001 원본(있으면) |
| 02_법규 | 개인정보보호법·산업안전보건법 등 업종 공통 |
| 03_해설서 | 조직 차원 표준 해설 (드문 경우) |
| 04_AsIs | **회사 미션·비전·조직도·전사 공통 POL·용어집** |
| 05_산업가이드 | 업종 프레임워크 (NIST, COBIT, 식약처 가이드) |

## 에이전트 동작
- 모든 분석 에이전트가 표준별 `_inputs/` 전에 이 폴더 선행 스캔
- 용어·조직·컨텍스트를 **모든 산출물에 일관되게** 반영
- 표준별 `_inputs/04_AsIs/` 가 우선, 이 폴더는 **배경 참조**
