# 06_TMP_템플릿 — 템플릿(TMP)

## 역할
실제 기록이나 산출물을 작성하기 위한 **빈 양식**.

## 특징 (엄격)
- **내용은 최소** — 입력 항목 구조화
- **샘플 값 절대 금지** — 샘플은 `07_EX_작성예시/` 로 분리
- 재사용 가능
- 문서명·작성자·작성일·버전·작성 필드·승인 항목·첨부 목록 등 포함

## 파일명 규칙
```
TMP-{기능}-{###}_{이름}_v{버전}.md
```
예: `TMP-IR-001_침해사고_내부신고서_v1.0.md`

## Frontmatter 필수
`type`, `doc_id`, `title`, `version`, `parent_wi`, `related_ex`

## 판단 기준
> "이 문서를 그대로 복사해서 실제 기록 작성에 사용할 수 있는가 → TMP"

## 생성 담당
Agent `wi-tmp-writer`

## 금지사항 (QA Fail 사유)
- ❌ 샘플 데이터 기입
- ❌ REC 와 같은 폴더에 공존
- ❌ 상위 WI 링크 없음

## 관련 폴더
- 작성예시(샘플 채움본): `07_EX_작성예시/`
- 실제 기록(운영 단계): `08_REC_기록/`
- 상위 업무지침: `05_WI_업무지침/`

## Obsidian 사용
Obsidian Templates 플러그인에서 삽입 가능. 하지만 **실제 기록은 08_REC 로**, 여기에 쌓지 않기.
