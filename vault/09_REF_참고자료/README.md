# 09_REF_참고자료 — 참고자료(REF)

## 역할
외부 규정·표준·가이드·참고 문헌 등을 **요약·정리**한 자료.

## 특징
- 관리체계의 **직접 통제 문서는 아님** (참고용)
- 외부 규제 또는 내부 참고 해설
- 원문(`_inputs/`)의 요지 정리본
- **없어도 체계는 운영되지만, 있으면 이해·적용이 쉬워지는** 수준

## 파일명 규칙
```
REF-{###}_{이름}_v{버전}.md
```
예: `REF-001_개인정보보호법_요약_v1.0.md`

## Frontmatter 필수
`type`, `doc_id`, `title`, `source`, `source_date`, `version`, `status`

## 포함 항목
- 요약
- 핵심 조항/요구사항
- 관련 내부 문서 (POL/PRO/WI 링크)
- 원문 인용 (라이선스 허용 시만)
- 변경 알림 (원문 개정 시 본 REF 재검토)

## 생성 담당
Agent `standard-analyzer` — 표준 편입 시 `_inputs/02_법규`, `03_해설서` 에서 발췌·요약

## 판단 기준
> "없어도 체계는 운영되지만 **있으면 이해와 적용이 쉬워지는 자료** 면 REF"

## 저작권 주의
- 공공저작물(법규)은 원문 인용 가능
- ISO/IEC 원문은 paraphrase 만 (직접 복사 금지)
- 해설서·가이드는 발행처 라이선스 확인 후
