# MAT-006 문서 계층 추적 매트릭스

> **공통 매트릭스**. 전사 모든 표준의 `POL → PRO → WI → TMP → EX` 5단 계층을 트리·표 2가지 뷰로 제공.
> 상위: [[MOC_추적성매트릭스]] · [[02_문서번호체계]] §MAT 번호 할당 원칙
> 유사 매트릭스와 역할 구분:
> - MAT-002 (규제요구사항 대조표): **법규 조항** 축
> - MAT-011~ (표준별 추적성): **Req-ID** 축
> - **MAT-006 (본 문서)**: **문서 계층(구조)** 축

## 목적

- 정책 1건이 실무 양식(TMP·EX) 까지 도달하는 **경로 완결성** 을 한눈에 확인
- **고아 문서** (상위 연결 없음) 와 **끊긴 경로** (미생성 하위) 를 탐지
- 고객사 배포·심사자 설명 시 "정책 → 실무" 흐름 시각화 자료로 활용
- 각 표준 편입 시 traceability-mapper 가 해당 표준 섹션을 append/갱신

## 범위

본 문서에 편입된 모든 표준. 표준 편입 시 §N 같은 구조로 섹션 추가.

## 상태 기호 범례

| 기호 | 의미 |
|---|---|
| ✅ | 경로 완전 (POL → PRO → WI → TMP → EX 전부 존재) |
| 🟩 | TMP 존재·EX 의도적 생략 (단순 양식, 원칙상 허용) |
| 🟡 | TMP 존재·EX 미작성 (운영 단계 보완 대상) |
| 🟦 | WI 존재·TMP 불필요 (순수 절차·지침형 WI) |
| ⚠️  | unresolved (future_wi / future_tmp 큐 등록됨) |
| ❌ | 고아 (상위 참조 없음) |
