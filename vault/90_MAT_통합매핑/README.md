# 90_MAT_통합매핑 — 매핑·관리대장(MAT)

## 역할
문서 간 관계·관리 상태·요구사항 커버리지를 **추적하는 통합 관리 문서**. 심사 대응과 체계 운영의 핵심.

## 수록 문서 (전사 공통 최대 10종 + 표준별)

| [[MAT-001_문서관리대장]] | 모든 POL/PRO/WI/TMP/EX/REC/REF 의 단일 인벤토리 |
| [[MAT-002_규제요구사항_대조표]] | 표준·법규 조항 ↔ POL/PRO/WI 매핑 |

## MAT 번호 구역 (상세: [[02_문서번호체계]] §MAT 번호 할당 원칙)
- **001~010**: 전사 공통 (최대 10종, 재배정 금지)
- **011~099**: 표준별 추적성 (순차)
- **100~**: 예약

## 관리 담당
Agent `traceability-mapper` — 표준 편입 시마다 공통 매트릭스(MAT-001·003·004·005·006) 최신화 + 해당 표준 추적성 신규 생성

## 특징
- 문서 자체보다 **관리성**이 목적
- 심사자의 진입점으로 설계
- Dataview 플러그인으로 자동 집계 가능 (MAT-001)

## 판단 기준
> "문서끼리의 관계나 상태를 관리하면 MAT"

- 🟡 작업중 (일부 누락)
- ⛔ 미착수 (매핑 없음)
