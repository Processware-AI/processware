# 90_MAT_통합매핑 — 매핑·관리대장(MAT)

## 역할
문서 간 관계·관리 상태·요구사항 커버리지를 **추적하는 통합 관리 문서**. 심사 대응과 체계 운영의 핵심.

## 수록 문서 (고정 5종 + 표준별)

### 전사 통합 MAT-001~005
| 문서 | 역할 |
|---|---|
| [[MAT-001_문서관리대장]] | 모든 POL/PRO/WI/TMP/EX/REC/REF 의 단일 인벤토리 |
| [[MAT-002_규제요구사항_대조표]] | 표준·법규 조항 ↔ POL/PRO/WI 매핑 |
| [[MAT-003_산출물_목록표]] | 표준별 필수 산출물 체크리스트 |
| [[MAT-004_RACI_통합표]] | 전사 PRO 별 책임 분담 집계 (Accountable 중복·누락 탐지) |
| [[MAT-005_심사증적_인덱스]] | 심사자용 표준 조항 ↔ 산출물 ↔ 증적 |

### 표준별 추적성
`MAT-06_{표준코드}_추적성_v{버전}.md` — 표준 1건당 1개
예: `MAT-06_ISO9001_추적성_v0.1.md`

## 파일명 규칙
```
MAT-{###}_{이름}_v{버전}.md
```

## 관리 담당
Agent `traceability-mapper` — 표준 편입 시마다 MAT-001~005 최신화 + 해당 표준 추적성 신규 생성

## 특징
- 문서 자체보다 **관리성**이 목적
- 심사자의 진입점으로 설계
- Dataview 플러그인으로 자동 집계 가능 (MAT-001)

## 판단 기준
> "문서끼리의 관계나 상태를 관리하면 MAT"

## 커버리지 범례
- ✅ 반영완료 (POL+PRO+WI+증적경로 모두 연결)
- 🟡 작업중 (일부 누락)
- ⛔ 미착수 (매핑 없음)
