# MAT-007 프로세스 카탈로그

> **용도**: 사용자가 절차 번호를 모르고도 자연어로 차원 2 실행을 시작할 수 있도록, PRO/WI 의 트리거 키워드·역할·이벤트를 인덱싱한 통합 카탈로그.

> **갱신 정책**:
> - 자동: `traceability-mapper` 가 차원 1 빌드 종료 시 본 파일 갱신 (write phase 직후, MAT-001/003/004/006 갱신과 함께).
> - 수동: `/do --rebuild-catalog` 로 수동 트리거.
> - 사람이 수정한 행은 `manual_override: true` 표기 시 자동 갱신 시 보존.

> **사용 주체**: `process-router` 에이전트가 본 파일을 1회 LLM 컨텍스트로 로드 후 자연어 → WI 매칭. 사람이 직접 사용할 때는 §"빠른 검색표" 또는 §"전체 인덱스" 참조.
