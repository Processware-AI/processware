# 01_구성원칙 — 최상위 체계 기준

## 역할
이 볼트의 **"헌법"**. 모든 산출물·에이전트 행동의 기준선.

## 수록 문서
- [[표준프로세스_구성원칙]] — 전사 표준 프로세스 체계 구축 방안

## 포함된 원칙
1. 표준 정합성 (IMS 통합, 표준 무관)
2. 계층적 구조화 (8종 유형)
3. 프로세스 접근 (PDCA·RACI·KPI)
4. 테일러링 가능
5. AI 자동화 (4차원 PDCA 에이전트, HITL)
6. 단일 소스 (SSoT — vault)
7. 감사 추적성 (source_citation + REC + trace)
8. 준거 표준 매핑 (ISO/IEC/KS)
9. 3-Tier 프로세스 아키텍처 (경영/핵심/지원)
10. 문서 유형 체계 (8종 + 엄격 분리 + 입력자료 원칙)
11. 거버넌스 (PCB · AI 에이전트 역할 분리)
12. 폐쇄 루프 (Plan→Do→Check→Act→Plan' 자동 순환)

## 관리 규칙
- **이 폴더에는 구성원칙 문서 1개만** 유지
- 다른 체계 문서는 `00_공통관리/` 로
- 구성원칙 개정은 **구조 변경 = v{n+1}.0**, 경미는 v1.n
- 모든 에이전트가 실행 시작 시 먼저 Read
